Calderbank-Shor-Steane (CSS) code

Name
Calderbank-Shor-Steane (CSS) code §
Description
An \((n,2^k,d)\) CSS code is usually denoted as \([[n,k,d]]\). Encoding is based on two binary linear codes, an \([n,k_1,d_1]\) code \(C_1\) and \([n,k_2,d_2]\) code \(C_2\) with \(C_1^\perp \subseteq C_2\), such that \(k=k_1+k_2-n\) and \(d=\min\{d_1,d_2\}\).
Protection
Detects errors on \(d-1\) qubits, corrects errors on \(\left\lfloor (d-1)/2 \right\rfloor\) qubits.
