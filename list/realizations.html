Here is a list of all classical and quantum codes that have been realized in experimental or real-world devices.
Name Realization(s)
Alamouti code Wireless standards since: 3G, LTE, LTE-Advanced, and 5G.WiFi standards: 802.11n, 802.11ad, 802.11ay, etc.
Bacon-Shor code Trapped-ion qubits: state preparation, logical measurement, and stabilizer measurement for nine-qubit Bacon-Shor code demonstrated on a 13-qubit device by M. Cetina and C. Monroe groups [1].
Balanced code Balanced length-eight code, known as a 6b/8b encoding, used for balancing direct current in a communications system [2]
Binary BCH code Satellite communication [3]
Binomial code Microwave cavities coupled to superconducting circuits: state transfer between a binomial codeword to another system [4], error-correction protocol nearly reaching break-even [5], and a teleported CNOT gate [6].
Bose–Chaudhuri–Hocquenghem (BCH) code DVDs, disk drives, and two-dimensional bar codes [7].
Calderbank-Shor-Steane (CSS) stabilizer code Fully homomorphic encryption [8].Entanglement purification protocols related to quantum key distribution (QKD) [9].
Cat code Four-legged (\(S=1\)) cat code has been realized in a superconducting circuit device [10]. This paper is the first to reach break-even error-correction, where the lifetime of a logical qubit is on par with the lifetime of the noisiest constituent of the system.
Classical Goppa code Initial version of the McEliece public-key cryptosystem [11][12] and its variation by Niederreiter [13] where the generator matrix is replaced by the parity check matrix. Some of these were proven to be insecure since the public key exposes algebraic structure of code [14].
Constant-weight code Radio-network frequency hopping [15].
Convolutional code A type of convolutional code used in Real-time Application networks [16].Mobile and radio communications (3G networks) use convolutional codes concatenated with Reed-Solomon codes to obtain suitable performance [17].A convolutional code with rate 1/2 was used for deep-space and satellite communication [18]
Covering code Data compression both with or without compression [19].Football-pool problem: finding the smallest number of bets on a set of matches needed to guarantee at least one bet has at most \(\rho\) errors [20][21].
Fibonacci string-net code NMR: Implementation of braiding-based Hamamard gate on two qubits [22].
Five-qubit perfect code NMR: Implementation of perfect error correcting code on 5 spin subsystem of labeled crotonic acid for quantum network benchmarking [23]. Single-qubit logical gates [24].Superconducting qubits [25].Trapped-ion qubits: non-transversal CNOT gate between two logical qubits, including rounds of correction and fault-tolerant primitives such as flag qubits and pieceable fault tolerance, on a 12-qubit device by Quantinuum [26].Nitrogen-vacancy centers in diamond: fault-tolerant single-qubit Clifford operations [27].
Floquet color code Plaquette stabilizer measurement realized on the IBM Falcon superconducting-qubit device [28]
Fountain code Designed for servers sending data to many recipients, such as during broadcasting or file distribution [29][30].
Gabidulin code Public-key cryptosystems [31].Digital watermarking. The Gabidulin code provides efficient correction against luminance tampering and image-slicing distortion due to the consistency of the rank against alterations such as column swapping [32].
Generalized RS (GRS) code Commonly used in mass storage systems such as CDs, DVDs, QR codes etc.Various cloud storage systems [33].Public-key cryptosystems generalizing those that used Goppa codes [11][12][13], some of which were proven to be insecure [14]. More recent works focus on methods to mask the algebraic structure using subcodes of GRS codes [31]. For example, a key-recovery attack was developed in Ref. [34] for a variant of masking method proposed in Ref. [35].
Golay code Used in the Voyager 1 and 2 spacecraft [36].Radio communications [37].
Gottesman-Kitaev-Preskill (GKP) code Motional degree of freedom of a trapped ion: GKP encoding realized with the help of post-selection [38][39], followed by realization of reduced form of GKP error correction, where displacement error syndromes are measured to one bit of precision using an ion electronic state [40].Microwave cavity coupled to superconducting circuits: reduced form of GKP error correction, where displacement error syndromes are measured to one bit of precision using an ancillary transmon [41].In signal processing, where the conjugate variables are time and frequency instead of position and momentum, GKP state position-state wavefunctions correspond to Dirac combs [42].
Hamming code Commonly used when error rates are very low, for example, computer RAM or integrated circuits [43].Hamming-code based matrix embedding used in steganography [44][45].
Heavy-hexagon code Logical state preparation and flag-qubit error correction realized in superconducting-circuit devices (specifically, fixed-frequency transmon qubit architectures) by IBM for \(d=2\) [46][47] and \(d=3\) [48].
Honeycomb Floquet code Plaquette stabilizer measurement realized on the IBM Falcon superconducting-qubit device [28]
Interleaved RS (IRS) code The cross-interleaved RS (CIRC), an IRS code using two shortened RS codes and two forms of interleaving, was used for compact discs (CDs) [49] (see Ref. [50], Sec. 5.6 and Ref. [51], Ch. 4).
Justesen code Generating small-bias sample spaces, i.e., probability distributions that parity functions cannot typically distinguish from the uniform distribution [52].
Kitaev surface code One cycle of syndrome readout on 19-qubit planar and 24-qubit toric codes realized in two-dimensional Rydberg atomic arrays [53]. Signatures of corresponding topological phase of matter detected in superconducting circuits [54] and two-dimensional Rydberg atomic arrays [55].
Lloyd-Slotine nine-mode code Optical network by the Furusawa group [56].
Locally recoverable code (LRC) An \((18,14,7)\) LRC code has beed used in the Windows Azure cloud storage system [57]; see also Sec. 31.3.1.2 in Ref. [58].
Low-density parity-check (LDPC) code 5G NR cellular communication for the traffic channel [59].WiMAX (IEEE 802.16e) [60].Satellite transmission of digital television [61].
Matching code Braiding of defects has been demonstrated for a five-qubit version of code [62].
Maximum distance separable (MDS) code The McEliece Public Key Cryptosystem [63] uses algebraic coding theory to secure communications against eavesdropping attack, in which private keys are generator matrices of linear codes, i.e., \(G\). Public Keys shared to outside world are scrambled and permutated versions of \(G\), i.e., \(G^\prime=SGP\). Data to be encrypted, \(u\), is multiplied by public key and added intentional errors \(e\), i.e., \(x=uG^\prime+e\). Upon receiving encrypted data, private key owner can apply inverse permutation \(P^{-1}\) to \(x\), decode the scrambled message given the presence of \(e\) errors, and finally unscramble to obtain \(u\). Security parameters of the system are \(n\) and \(e\), hence MDS codes, such as GRS codes may provide the same security level for shorter key sizes, and the private key owner can decode arguably fast enough using known decoding algorithms.Automatic repeat request (ARQ) data transmission protocols ([51], Ch. 7).
Maximum-rank distance (MRD) code Useful for error and erasure correction in network coding [64][65].
Monitored random-circuit code Measurement induced quantum phases have been realized in a trapped-ion processor [66].
Niset-Andersen-Cerf code Realized in Ref. [67] in an optical system with 3 beam-splitters. The fidelity peaked around \(0.6\) for deterministic approach, and around \(0.77\) for the probabilistic approach (with a 25% chance of error).
One-hot code The bi-quinary code, a combination of one-hot 1-in-2 and 1-in-5 one-hot codes to encode decimal digits, was used in several early computers ([68], Ch. 27).Marking the state of a finite automaton [69].Used in machine-learning based classification tasks because one-hot encodings, as opposed to integer encodings, do not presume an order [70].
Pair-cat code Microwave cavities coupled to superconducting circuits by the Wang group [71].
Parity-check code Can be realized on almost every communication device. Parity-check codes are some of the earlier error-correcting codes ([68], Ch. 27).
Polar code Code control channels for the 5G NR (New Radio) interfaces [72].
Quadrature PSK (QPSK) Japanese and North American digital cellular and personal systems [73].
Quantum Tanner code Used to obtain explicit lower bounds in the sum-of-squares game [74].States that, on average, achieve small violations of check operators for quantum Tanner codes require a circuit of non-constant depth to make. They are used in the proof [75] of the No low-energy trivial states (NLTS) conjecture [76].
Quantum parity code (QPC) The \([[m^2,1,m]]\) codes for \(m\leq 7\) have been realized in trapped-ion quantum devices [77].
Quantum repetition code NMR: 3-qubit phase-flip code [78][79], with up to two rounds of error correction in liquid-state NMR [80].Superconducting circuits: 3-qubit phase-flip and bit-flip code by Schoelkopf group [81]; 3-qubit bit-flip code [82]; 3-qubit phase-flip code up to 3 cycles of error correction [83]; IBM 15-qubit device [84]; IBM Rochester device using 43-qubit code [85]; Google system performing up to 8 error-correction cycles on 5 and 9 qubits [86]; Google Quantum AI Sycamore utilizing up to 11 physical qubits and running 50 correction rounds [87]; Google Quantum AI Sycamore utilizing up to 25 qubits for comparison of logical error scaling with a quantum code [88]. Continuous error correction protocols have been implemented on a 3-qubit device [89].Semiconductor spin-qubit devices: 3-qubit devices at RIKEN [90] and Delft [91].Nitrogen-vacancy centers in diamond: 3-qubit phase-flip code [92][93] (see also Ref. [94]).Trapped-ion device: 3-qubit phase-flip algorithm implemented in 3 cycles on high fidelity gate operations [95].
Random code Distributed storage systems [96].
Rank-modulation code Electronic devices where charges can either increase in an individual cell or decrease in a block of adjacent cells, e.g., flash memories [97].
Raptor (RAPid TORnado) code Two versions of Raptor codes have been standardized by IETF: R10 and the more recent RaptorQ. RaptorQ is used in mobile multimedia broadcasts as specified in ETSI technical specifications. It is also used in the mobile Next Gen TV standard.Raptor codes are useful in scenarios where erasure (i.e. weak signal or noisy channel) is common, such as in military or disaster scenarios.
Reed-Muller (RM) code Deep-space communication, including the Mariner 9 spacecraft [98][99].
Reed-Solomon (RS) code RS Product Code (RSPC) was used in DVDs (see Ref. [51], Ch. 4).DSL technologies and their variants against impluse noise [100].Cryptographic primitives based on the hardness of decoding RS codes for more than \(1-\sqrt{k/n}+\epsilon\) errors. This is equivalent to the polynomial reconstruction problem [101].RS codes as outer codes concatenated with convolutional codes are used indirectly in space exploration programs such as Voyager and Galileo. RS codes were part of a temetry channel coding standard issued by the Consultative Committee for Space Data Systems (see Ref. [51], Ch. 3).The ubiquity of RS codes has yielded off-the-shelf VLSI intergrated-circuit decoding hardware [102] (see also Ref. [51], Ch. 5 and 10).Automatic repeat request (ARQ) data transmission protocols (see Ref. [51], Ch. 7).Slow-frequency-hop spread-spectrum transmission (see Ref. [51], Chs. 8-9).Coded sharding designs in blockchains to increase efficiency [103].Private Information Retrieval [104].Used in QR-Codes to retrieve damaged barcodes [105].Wireless communication systems such as 3G, DVB, and WiMAX [106].Correcting pooled testing results for SARS-CoV-2 [107].
Repetition code Repetition codes, in conjunction with other codes, were used in magnetic disks [108].
Residue AG code Improvements over the McEliece public-key cryptosystem to linear AG codes on curves of arbitrary genus [109]. Only the subfield subcode proposal remains resilient to attacks ([58], Sec. 15.7.5.3).Algebraic secret-sharing schemes [110].
Shor \([[9,1,3]]\) code Trapped-ion qubits: state preparation with 98.8(1)% and 98.5(1)% fidelity for state \(|\overline{0}\rangle\) and \(|\overline{1}\rangle\), respectively, by N. Linke group [77]. Variants of the code to handle coherent noise studied and realized by K. Brown and C. Monroe groups [111].Optical systems: quantum teleportation of information implemented by J.-W. Pan group on maximally entangled pair of one physical and one logical qubit with fidelity rate of up to 78.6% [112]. All-photonic quantum repeater architecture tested on the same code [113].
Skew-cyclic code Not directly implemented, but BCH codes form a subclass, and are used in DVD, solid state drive storage, etc.
Spacetime block code (STBC) High data-rate wireless communication, e.g., WiMAX (IEEE 802.16m) [114][115][116].
Steane \([[7,1,3]]\) code Trapped-ion qubits: seven-qubit device in Blatt group [117], ten-qubit QCCD device by Quantinuum [118] (see APS Physics Synopsys [119]). Fault-tolerant universal two-qubit gate set by Monz group [120]. Logical CNOT gate between two logical qubits, including rounds of correction and fault-tolerant primitives such as flag qubits and pieceable fault tolerance, on a 20-qubit device by Quantinuum [26]; logical fidelity interval of the combined preparation-CNOT-measurement procedure was higher than that of the unencoded physical qubits.Rydberg atom arrays: Lukin group [53].
Surface-17 code Implemented at ETH Zurich by the Wallraff group [121] and on the Zuchongzhi 2.1 superconducting quantum processor [122]. Both experimental error rates are above the pseudothreshold for this code relative to a single qubit. See Physics viewpoint for a summary [123].
Tensor-product code Construction can be used in magnetic recording by taking the tensor product of a Reed-Solomon code and a parity-check code [124].
Ternary Golay Code Code used in football pools with at least one good bet [125][21]. In fact, the code was originally constructed by Juhani Virtakallio and published in the Finnish football pool magazine Veikkaaja [21][126].
Two-legged cat code Lindbladian-based [127][128] and Hamiltonian-based 'Kerr-cat' [129] encodings have been achieved in superconducting circuit devices by the Devoret group; Ref. [128] also demonstrated a displacement-based gate. The Lindbladian-based scheme has further achieved a suppression of bit-flip errors that is exponential in the average photon number [130] up to a bit-flip time of 1ms. A bit-flip time of 1s has been achieved in a similar system in the classical bit regime [131].Approximate version realized in a superconducting circuit device by the Wang group [132].
Weight-two code Two-in-five, also known as the two-out-of-five code, was used in the United States Postal Service's POSTNET barcode system as well as the Postal Alpha-numeric Encoding Technique (PLANET).Two-in-five code forms the numerical part of the Code 39 barcode encoding.Two-in-five code was used on early IBM computers [133][126].
XZZX surface code Distance-five 25-qubit code implemented on a superconducting quantum processor by Google Quantum AI [88]. This code outperformed the average of several instances of the smaller distance-three 9-qubit \(XZZX\) variant of the surface-17 code realized on the same device, both in terms of logical error probability over 25 cycles and in terms of logical error per cycle. This increase in error-correcting capabilities while using more physical qubits supports the notion of an error threshold. Braiding of defects has been demonstrated for this code [134].
Zetterberg code Code used to provide better protection of data transmission with its double error correcting capacity [135].
\([[4,2,2]]\) CSS code \([[4,1,2]]\) subcode implemented using four-qubit graph state of photons [136]. Trapped-ion device by IonQ [137].Logical state preparation and flag-qubit error correction realized in superconducting-circuit devices by IBM [46][47].The subcode \(\{|\overline{00}\rangle,|\overline{10}\rangle\}\) [138] and \(\{|\overline{00}\rangle,|\overline{01}\rangle\}\) [87], treated as a planar surface code, has been realized in superconducting-circuit devices.Logical gates between two copies of the subcode \(\{|\overline{10}\rangle,|\overline{11}\rangle\}\), interpreted as lattice surgery between planar surface codes, realized in superconducting circuits [139].Logical gates for the \(\{|\overline{00}\rangle,|\overline{11}\rangle\}\) subcode, treated as a planar code, realized in superconducting circuits [140].
