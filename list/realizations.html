Here is a list of all classical and quantum codes that have been realized in experimental or real-world devices.
Name Realization(s)
Bacon-Shor code Trapped-ion qubits: state preparation, logical measurement, and stabilizer measurement for nine-qubit Bacon-Shor code demonstrated on a 13-qubit device by M. Cetina and C. Monroe groups [1].
Binary Golay code Used in the Voyager 1 and 2 spacecraft [2].
Binary repetition code Repetition codesm in conjunction with other codes, were used in magnetic disks [3].
Binomial code Realized in microwave cavities coupled to superconducting circuits [4].
Cat code Two-legged (\(S=0\)) Lindbladian-based [5][6] and Hamiltonian-based 'Kerr-cat' encoding [7] has been achieved in superconducting circuit devices by the Devoret group; Ref. [6] also demonstrated a displacement-based gate. The Lindbladian-based scheme has further achieved a suppression of bit-flip errors that is exponential in the average photon number [8] up to a bit-flip time of 1 ms. A bit-flip time of up to 1 sec has been achieved while away from the logical-qubit regime [9].Four-legged (\(S=1\)) cat code has been realized in a superconducting circuit device [10]. This paper is the first to reach break-even error-correction, where the lifetime of a logical qubit is on par with the lifetime of the noisiest constituent of the system.Approximate version of the \(S=0\) code realized in a superconducting circuit device by the Wang group [11].
Convolutional code A type of convolutional code used in Real-time Application networks [12].Mobile and radio communications (3G networks) use convolutional codes concatenated with Reed-Solomon codes to obtain suitable performance [13].A convolutional code with rate 1/2 was used for deep-space and satellite communication [14]
Covering code Data compression both with or without compression [15].Football-pool problem: finding the smallest number of bets on a set of matches needed to guarantee at least one bet has at most \(\rho\) errors [16][17].
Fountain code Designed for servers sending data to many recipients, such as during broadcasting or file distribution [18].
Generalized Reed-Solomon (GRS) code Various cloud storage systems [19].Public-key cryptosystems [20]. Initial construction of McEliece Public Key Cryptosystem was based on Goppa codes which are subfield subcode of GRS codes. Public Key Cryptosystem designs based on GRS codes first proposed in Ref. [21], which replaced the generator matrix with the parity check matrix, were proven to be insecure [22] since the public key exposes algebraic structure of code. More recent works focus on methods to mask the algebraic structure using subcodes of GRS codes [20]. Lately a key-recovery attack was developed in [23] for a variant of masking method proposed by [24].
Goppa code The binary version \( (q=2) \) is commonly used in post-quantum cryptosystems such as the McElise cryptosystem [25].
Gottesman-Kitaev-Preskill (GKP) code GKP encoding realized in the motional degree of freedom of a trapped ion with the help of post-selection [26][27], followed by realization of dissipative stabilization scheme [28].A reduced form of GKP error correction, measuring displacement error syndromes to one bit of precision using an ancillary transmon, realized in a microwave cavity coupled to superconducting circuits [29].
Hamming code Commonly used when error rates are very low, for example, computer RAM or integrated circuits [30].
Heavy-hexagon code Logical state preparation and flag-qubit error correction realized in superconducting-circuit devices (specifically, fixed-frequency transmon qubit architectures) by IBM for \(d=2\) [31][32] and \(d=3\) [33].
Homological bosonic code No experimental realization. However, Ref. [34] describes a potential experimental optical procedure for the simplest non-trival code with 5 modes.
Justesen code Generating small-bias sample spaces, i.e., probability distributions that parity functions cannot typically distinguish from the uniform distribution [35].
Kitaev surface code One cycle of syndrome readout on 19-qubit planar and 24-qubit toric codes realized in two-dimensional Rydberg atomic arrays [36]. Signatures of corresponding topological phase of matter detected in superconducting circuits [37] and two-dimensional Rydberg atomic arrays [38].
Lloyd-Slotine nine-mode code Optical network by the Furusawa group [39].
Locally recoverable code (LRC) An \((18,14,7)\) LRC code has beed used in the Windows Azure cloud storage system [40]; see also Sec. 31.3.1.2 in Ref. [41].
Low-density parity-check (LDPC) code 5G NR cellular communication for the traffic channel [42].WiMAX (IEEE 802.16e) [43].Satelite transmission of digital television [44].
Maximum distance separable (MDS) code The McEliece Public Key Cryptosystem [45] uses algebraic coding theory to secure communications against eavesdropping attack, in which private keys are generator matrices of linear codes, i.e., \(G\). Public Keys shared to outside world are scrambled and permutated versions of \(G\), i.e., \(G^\prime=SGP\). Data to be encrypted, \(u\), is multiplied by public key and added intentional errors \(e\), i.e., \(x=uG^\prime+e\). Upon receiving encrypted data, private key owner can apply inverse permutation \(P^{-1}\) to \(x\), decode the scrambled message given the presence of \(e\) errors, and finally unscramble to obtain \(u\). Security parameters of the system are \(n\) and \(e\), hence MDS codes, such as GRS codes may provide the same security level for shorter key sizes, and the private key owner can decode arguably fast enough using known decoding algorithms.
Maximum-rank distance (MRD) code Useful for error and erasure correction in network coding [46][47].
Monitored random-circuit code Measurement induced quantum phases have been realized in a trapped-ion processor [48].
Niset-Andersen-Cerf code Realized in Ref. [49] in an optical system with 3 beam-splitters. The fidelity peaked around \(0.6\) for deterministic approach, and around \(0.77\) for the probabilistic approach (with a 25% chance of error).
Perfect quantum code NMR: Implementation of identity, NOT, and Haddamard gates along with perfect error correcting code on single qubit errors in 5 qubit circuit [50].NMR: Implementation of perfect error correcting code on 5 spin subsystem of labeled crotonic acid for quantum network benchmarking [51].
Polar code Code control channels for the 5G NR (New Radio) interfaces [52].
Quantum Tanner code Used to obtain explicit lower bounds in the sum-of-squares game [53].
Quantum parity code (QPC) The \([[m^2,1,m]]\) codes for \(m\leq 7\) have been realized in trapped-ion quantum devices [54].Non-determinisitic linear-optical encoding [55] whose success probability \(P_{E}\) is determined by the efficiency \(\eta\) of the photonic encoding circuit. A threshold \(\eta > 0.82 \) exists for the efficiency, above which \(P_{E}\to 1\) as \(m_1\to\infty\) given particular \(m_2\).Studied in the context of error-corrected quantum repeaters [56].
Quantum repetition code NMR: 3-qubit phase-flip code [57][58], with up to two rounds of error correction in liquid-state NMR [59].Superconducting circuits: 3-qubit phase-flip and bit-flip code by Schoelkopf group [60], IBM 15-qubit device [61], Google experiment performing up to 8 error-correction cycles on 5 and 9 qubits [62], and Google Quantum AI Sycamore utilizing from 5 to 21 qubits [63]. Continuous error correction protocols have been implemented on a 3-qubit superconducting circuit device [64].Semiconductor spin-qubit devices: 3-qubit devices at RIKEN [65] and Delft [66].Nitrogen-vacancy centers in diamond: 3-qubit device [67].See Table S6 in Ref. [63] for a history of earlier implementations.Trapped-ion device: 3-qubit phase-flip algorithm implemented in 3 cycles on high fidelity gate operations [68].NV Center: 3-qubit phase-flip code [69].
Random code Distributed storage systems [70].
Rank-modulation code Electronic devices where charges can either increase in an individual cell or decrease in a block of adjacent cells, e.g., flash memories [71].
Raptor (RAPid TORnado) code Two versions of Raptor codes have been standardized by IETF: R10 and the more recent RaptorQ. RaptorQ is used in mobile multimedia broadcasts as specified in ETSI technical specifications. It is also used in the mobile Next Gen TV standard.Raptor codes are useful in scenarios where erasure (i.e. weak signal or noisy channel) is common, such as in military or disaster scenarios.
Reed-Muller (RM) code Deep-space communication, including the Mariner 9 spacecraft [72][73].
Reed-Solomon (RS) code Cross-interleaved RS code (CIRC) is adopted in Compact Discs (CDs) and RS Product Code (RSPC) in DVDs; see Ch. 4 of Ref. [74].In DSL technologies and its variants against impluse noise [75].RS codes as outer codes concatenated with convolutional codes are used indirectly in solar exploration programs; see Ch. 3 of Ref. [74].Coded sharding designs in blockchains to increase efficiency [76].Private Information Retrieval [77].Used in QR-Codes to retrieve damaged barcodes [78].Wireless communication systems such as 3G, DVB, and WiMAX [79].Correcting pooled testing results for SARS-CoV-2 [80].
Shor \([[9,1,3]]\) code Trapped-ion qubits: state preparation with 98.8(1)% and 98.5(1)% fidelity for state \(|\overline{0}\rangle\) and \(|\overline{1}\rangle\), respectively, by N. Linke group [54]. Variants of the code to handle coherent noise studied and realized by K. Brown and C. Monroe groups [81].All-photonic quantum repeater architecture [82].
Single parity-check code Can be realized on almost every communication device.
Skew-cyclic code Not directly implemented, but BCH codes form a subclass, and are used in DVD, solid state drive storage, etc.
Spacetime block code (STBC) High data-rate wireless communication, e.g., WiMAX (IEEE 802.16m) [83][84].
Steane \([[7,1,3]]\) code Trapped-ion qubits: seven-qubit device in Blatt group [85], ten-qubit QCCD device by Quantinuum [86], fault-tolerant universal two-qubit gate realized by Monz group [87].Rydberg atom arrays: Lukin group [36].
Surface-17 code Implemented at ETH Zurich by the Wallraff group [88] and on the Zuchongzhi 2.1 superconducting quantum processor [89]. Both experimental error rates are above the pseudothreshold for this code relative to a single qubit.
Zetterberg code Code used to provide better protection of data transmission with its double error correcting capacity [90].
\([[4,2,2]]\) CSS code \([[4,1,2]]\) subcode implemented using four-qubit graph state of photons [91]. Trapped-ion device by IonQ [92].Logical state preparation and flag-qubit error correction realized in superconducting-circuit devices by IBM [31][32].The subcode \(\{|\overline{00}\rangle,|\overline{10}\rangle\}\) [93] and \(\{|\overline{00}\rangle,|\overline{01}\rangle\}\) [63], treated as a planar surface code, has been realized in superconducting-circuit devices.Logical gates between two copies of the subcode \(\{|\overline{10}\rangle,|\overline{11}\rangle\}\), interpreted as lattice surgery between planar surface codes, realized in superconducting circuits [94].Logical gates for the \(\{|\overline{00}\rangle,|\overline{11}\rangle\}\) subcode, treated as a planar code, realized in superconducting circuits [95].
\([[5,1,3]]\) perfect code First realized in NMR [51].Demonstration with superconducting qubits [96].
