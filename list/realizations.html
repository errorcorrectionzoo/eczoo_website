Here is a list of all classical and quantum codes that have been realized in experimental or real-world devices.
Name Realization(s)
Bacon-Shor code Trapped-ion qubits: state preparation, logical measurement, and stabilizer measurement for nine-qubit Bacon-Shor code demonstrated on a 13-qubit device by M. Cetina and C. Monroe groups [1].
Binary repetition code Repetition codes were used in magnetic disks.
Binomial code Realized in microwave cavities coupled to superconducting circuits [2].
Cat code Two-legged (\(S=1\)) cat code has been realized by two superconducting cavities coupled through a Josephson junction by Devoret group [3]. Exponential suppression of bit-flip errors achieved [4] up to a bit-flip time of 1 ms.Four-legged (\(S=2\)) cat code has been realized in superconducting circuit [5]. This paper is the first break-even point of quantum error-correction, where the lifetime of a logical qubit is on par with the lifetime of the noisiest constituent of the system.
Convolutional code A type of convolutional code used in Real-time Application networks [6].Mobile and radio communications (3G networks) use convolutional codes concatenated with Reed-Solomon codes to obtain suitable performance [7].A convolutional code with rate 1/2 was used for deep-space and satellite communication [8]
Goppa code The binary version \( (q=2) \) is commonly used in post-quantum cryptosystems such as the McElise cryptosystem [9].
Gottesman-Kitaev-Preskill (GKP) code Microwave cavity coupled to superconducting circuits [10].Motional degree of freedom in a trapped ion [11].
Haah cubic code Numerical simulations showed that \(T_{\rm{mem}}(\beta) \ge e^{c \beta^2/12}\), suggesting the possibility of macroscopically feasible memory times [12].
Hamming code Commonly used when error rates are very low, for example, computer RAM.
Heavy-hexagon code Logical state preparation and flag-qubit error correction realized in superconducting-circuit devices (specifically, fixed-frequency transmon qubit architectures) by IBM for \(d=2\) [13][14] and \(d=3\) [15].
Homological bosonic code No experimental realization. However, Ref. [16] describes a potential experimental optical procedure for the simplest non-trival code with 5 modes.
Justesen code Generating small-bias sample spaces, i.e., probability distributions that parity functions cannot typically distinguish from the uniform distribution [17].
Kitaev surface code Distance-two surface codes have been implemented by Andersen et al. [18], Erhard et al. [19], Marques et al. [20], Google Quantum AI [21], and both planar and toric versions by Bluvstein et al. [22]. Distance-three surface code implemented at ETH Zurich [23] and on the Zuchongzhi 2.1 superconducting quantum processor [24]. Signatures of corresponding topological phase of matter detected in superconducting circuits [25] and two-dimensional arrays of Rydberg atoms [26].
Low-density parity-check code Used in 5G NR cellular communication for the traffic channel [27].WiMAX (IEEE 802.16e) [28].Satelite transmission of digital television [29].
Monitored random-circuit code Measurement induced quantum phases have been realized in a trapped-ion processor [30].
Niset-Andersen-Cerf code Realized in Ref. [31] in an optical system with 3 beam-splitters. The fidelity peaked around \(0.6\) for deterministic approach, and around \(0.77\) for the probabilistic approach (with a 25% chance of error).
Polar code Code control channels for the 5G NR (New Radio) interfaces.
Quantum parity code (QPC) The \([[m^2,1,m]]\) codes for \(m\leq 7\) have been realized in trapped-ion quantum devices [32].Non-determinisitic linear-optical encoding [33] whose success probability \(P_{E}\) is determined by the efficiency \(\eta\) of the photonic encoding circuit. A threshold \(\eta > 0.82 \) exists for the efficiency, above which \(P_{E}\to 1\) as \(m_1\to\infty\) given particular \(m_2\).Studied in the context of error-corrected quantum repeaters [34].
Quantum repetition code Superconducting circuits: IBM 15-qubit [35], Google Quantum AI Sycamore [21].3-qubit semiconductor spin-qubit computer at RIKEN [36] and Delft [37].3-qubit codes can be used to benchmark device performance [38].See Table S6 in Ref. [21] for a history of earlier implementations.
Random code Distributed storage systems [39].
Rank-modulation code Electronic devices where charges can either increase in an individual cell or decrease in a block of adjacent cells, e.g., flash memories [40].
Reed-Solomon (RS) code Numerous applications: computer hard-disks, DVDs, digital audio tapes, 3G telephone networks (IS-2000, Release D), NASA deep-space exploration, etc.Correcting pooled testing results for SARS-CoV-2 [41].
Shor \([[9,1,3]]\) code Trapped-ion qubits: state preparation with 98.8(1)% and 98.5(1)% fidelity for state \(|\overline{0}\rangle\) and \(|\overline{1}\rangle\), respectively, by N. Linke group [32]. Variants of the code to handle coherent noise studied and realized by K. Brown and C. Monroe groups [42].All-photonic quantum repeater architecture [43].
Single parity-check code Can be realized on almost every communication device.
Skew-cyclic code Not directly implemented, but BCH codes form a subclass, and are used in DVD, solid state drive storage, etc.
Spacetime block code (STBC) High data-rate wireless communication, e.g., WiMAX (IEEE 802.16m) [44][45].
Steane \([[7,1,3]]\) code Trapped-ion qubits: seven-qubit device in Blatt group [46] and ten-qubit QCCD device by Quantinuum [47].Rydberg atom arrays: Lukin group [22].
Vector code Useful for error and erasure correction in network coding
Zetterberg code Code used to provide better protection of data transmission with its double error correcting capacity [48].
\([[4,2,2]]\) CSS code Trapped-ion device by IonQ [49].Logical state preparation and flag-qubit error correction realized in superconducting-circuit devices by IBM [13][14].
\([[5,1,3]]\) perfect code First realized in NMR [50].Demonstration with superconducting qubits [51].
