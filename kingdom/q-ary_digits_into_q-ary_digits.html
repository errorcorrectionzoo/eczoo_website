Welcome to the Galois-field Kingdom.

Galois-field \(q\)-ary code Encodes \(K\) states (codewords) in \(n\) \(q\)-ary coordinates over the field \(GF(q)=\mathbb{F}_q\) and has distance \(d\). Usually denoted as \((n,K,d)_q\). The distance is the minimum number of coordinates where two strings in the code differ. Protection: Detects errors on up to \(d-1\) coordinates, corrects erasure errors on up to \(d-1\) coordinates, and corrects general errors on up to \(\left\lfloor (d-1)/2 \right\rfloor\) coordinates. Parents: Error-correcting code (ECC). Parent of: Additive \(q\)-ary code, Algebraic-geometry (AG) code, Cyclic \(q\)-ary code.
Additive \(q\)-ary code A \(q\)-ary code whose codewords are closed under addition, i.e., for any codewords \(x,y\), \(x+y\) is also a codeword. Parents: Galois-field \(q\)-ary code. Parent of: Linear \(q\)-ary code. Cousins: Stabilizer code over \(GF(4)\). Cousin of: Galois-qudit stabilizer code, Stabilizer code over \(GF(q^2)\).
Algebraic-geometry (AG) code Stub. Parents: Galois-field \(q\)-ary code. Parent of: Evaluation AG code, Residue AG code.
Cyclic \(q\)-ary code A \(q\)-ary code of length \(n\) is cyclic if, for each codeword \(c_1 c_2 \cdots c_n\), the cyclically shifted string \(c_n c_1 \cdots c_{n-1}\) is also a codeword. Parents: Cyclic code, Galois-field \(q\)-ary code. Parent of: \(q\)-ary duadic code. Cousin of: Reed-Solomon (RS) code.
Generalized Reed-Solomon (GRS) code[1] Stub. Parents: Alternant code, Evaluation AG code, Distributed-storage code. Parent of: Bose–Chaudhuri–Hocquenghem (BCH) code, Goppa code, Reed-Solomon (RS) code. Cousin of: Quantum maximum-distance-separable (MDS) code.
Reed-Muller (RM) code[2][3][4] Member of the RM\((r,m)\) family of linear binary codes derived from multivariate polynomials. The code parameters are \([2^m,\sum_{j=0}^{r} {m \choose j},2^{m-r}]\), where \(r\) is the order of the code satisfying \(0\leq r\leq m\). Punctured RM codes RM\(^*(r,m)\) are obtained from RM codes by deleting one or more coordinates from each codeword. Parents: Bose–Chaudhuri–Hocquenghem (BCH) code, Quaternary code over \(\mathbb{Z}_4\), Divisible code, Binary duadic code, Group code. Parent of: Binary Hamming code. Cousins: Dual linear code, Cyclic binary code. Cousin of: Hadamard code, Majorana stabilizer code, Polar code, Quantum Reed-Muller code, Quantum divisible code, Simplex code.
Linear \(q\)-ary code An \((n,K,d)_q\) linear code is denoted as \([n,k,d]_q\), where \(k=\log_{q}K\) need not be an integer. Its codewords form a linear subspace, i.e., for any codewords \(x,y\), \(\alpha x+ \beta y\) is also a codeword for any \(q\)-ary digits \(\alpha,\beta\). Protection: Distance \(d\) of a linear code is the number of nonzero entries in the (nonzero) codeword with the smallest such number. Corrects any error set such that the difference of any pair of distinct elements of the set is a codeword. Parents: Additive \(q\)-ary code. Parent of: Alternant code, Gabidulin code, Group code, Maximum distance separable (MDS) code, Projective \(q\)-ary code, Two-weight code, Wozencraft ensemble code, \(q\)-ary duadic code. Cousin of: Entanglement-assisted (EA) QECC, Entanglement-assisted (EA) stabilizer code, Galois-qudit CSS code, Low-density parity-check (LDPC) code, Modular-qudit CSS code, True Galois-qudit stabilizer code.
Gabidulin code[5][6] Also called a vector rank-metric code. A linear code over \(GF(q^N)\) that corrects errors over rank metric instead of the traditional Hamming distance. Every element \(GF(q^N)\) can be written as an \(N\)-dimensional vector with coefficients in \(GF(q)\), and the rank of a set of elements is rank of the matrix formed by their coefficients. Protection: Set of vectors \(\{x_1, x_2, \ldots, x_M\}\) determines a rank code with distance \(d=\min d(x_i, x_j)\). The code with distance \(d\) corrects all errors with rank of the error not greater than \(\lfloor (d-1)/2\rfloor\). Parents: Linear \(q\)-ary code, Rank-metric code. Cousins: Maximum-rank distance (MRD) code.
Maximum distance separable (MDS) code[7] A \([n,k,d]_q\) \(q\)-ary linear code is an MDS code if parameters \(n\), \(k\), \(d\), and \(q\) are such that the Singleton bound \begin{align} d \leq n-k+1 \end{align} becomes an equality. A code is called almost MDS (AMDS) when \(d=n-k\). A bound for general \(q\)-ary codes can also be formulated; see Thm. 1.9.10 in Ref. [8]. Protection: Given \(n\) and \(k\), MDS codes have the highest distance possible of all codes and so have the best possible error correction properties. Parents: Linear \(q\)-ary code, Locally recoverable code (LRC). Parent of: Reed-Solomon (RS) code. Cousin of: Matrix computation code, Maximum-rank distance (MRD) code, Quantum maximum-distance-separable (MDS) code.
\(q\)-ary duadic code[9][10][11][12] Member of a pair of cyclic linear binary codes that satisfy certain relations, depending on whether the pair is even-like or odd-like duadic. Duadic codes exist only when \(q\) is a square modulo \(n\) [10]. Parents: Linear \(q\)-ary code, Cyclic \(q\)-ary code. Parent of: \(q\)-ary quadratic-residue (QR) code. Cousin of: Binary duadic code.
Bose–Chaudhuri–Hocquenghem (BCH) code[13] Stub. Parents: Generalized Reed-Solomon (GRS) code, Cyclic binary code. Parent of: Binary Hamming code, Reed-Muller (RM) code. Cousin of: Reed-Solomon (RS) code, Skew-cyclic code.
Goppa code[14][15][16] Let \( G(z) \) be a polynomial describing a projective plane curve with coefficients from \( GF(q^m) \) for some fixed integer \(m\). Let \( L \) be a finite subset of the extension field \( GF(q^m) \) where \(q\) is prime, meaning \( L = \{\alpha_1, \cdots, \alpha_n\} \) is a subset of nonzero elements of \( GF(q^m) \). A Goppa code \( \Gamma(L,G) \) is an \([n,k,d]\) linear code consisting of all vectors \(a = a_1, \cdots, a_n\) such that \( R_a(z) =0 \) modulo \(G(z)\), where \( R_a(z) = \sum_{i=1}^n \frac{a_i}{z - \alpha_i} \). Protection: The length \( n = |L| \) , dimension \( k \geq n-mr \) where \( r = \text{deg} G(z) \), and the minimum distance \( d \geq r +1 \). Parents: Generalized Reed-Solomon (GRS) code, Residue AG code. Cousin of: Binary quantum Goppa code.
Reed-Solomon (RS) code[17][18] An \([n,k,n-k+1]_q\) linear code based on polynomials over \(GF(q)\). Let \(\{\alpha_1,\cdots,\alpha_n\}\) be \(n\) distinct nonzero elements of \(GF(q)\) with \(q>n\). An RS code encodes \(\mu=\{\mu_0,\cdots,\mu_{k-1}\}\) into \(\{f_\mu(\alpha_1),\cdots,f_\mu(\alpha_n)\}\), with polynomial \begin{align} f_\mu(x)=\mu_0+\mu_1 x + \cdots + \mu_{k-1}x^{k-1}. \end{align} In other words, each codeword \(\mu\) is a string of values of the corresponding polynomial \(f_\mu\) at the points \(\alpha_i\). Protection: Since each polynomial \(f_{\mu}\) is of degree less than \(k\), it can be determined from its values at \(k\) points. This means that RS codes can correct erasures on up to \(n-k\) registers. The resulting distance, \(d=n-k+1\), saturates the Singleton bound. Parents: Generalized Reed-Solomon (GRS) code, Maximum distance separable (MDS) code. Parent of: Extended RS code, Folded RS code. Cousins: Binary Hamming code, Bose–Chaudhuri–Hocquenghem (BCH) code, Cyclic \(q\)-ary code. Cousin of: Approximate secret-sharing code, Convolutional code, Galois-qudit polynomial code (QPyC), Justesen code, Maximum-rank distance (MRD) code, Prime-qudit polynomial code (QPyC), Rank-modulation code.
Binary Hamming code[19] An infinite family of perfect linear codes with parameters \((2^r-1,2^r-r-1, 3)\) for \(r \geq 2\). Their \(r \times (2^r-1) \) parity check matrix \(H\) has all possible non-zero \(r\)-bit strings as its columns. Protection: Can detect 1-bit and 2-bit errors, and can correct 1-bit errors. Parents: Linear binary code, Perfect code, Reed-Muller (RM) code, Bose–Chaudhuri–Hocquenghem (BCH) code. Cousins: Binary quadratic-residue (QR) code, \([[2^r, 2^r-r-2, 3]]\) quantum Hamming code. Cousin of: Binary repetition code, Hadamard code, Reed-Solomon (RS) code, Simplex code, Steane \([[7,1,3]]\) code, \([[2^r-1, 2^r-2r-1, 3]]\) Hamming-based CSS code.
Evaluation AG code Stub. Parents: Algebraic-geometry (AG) code. Parent of: Generalized Reed-Solomon (GRS) code. Cousin of: Residue AG code.
Residue AG code Also called a differential code. Stub. Parents: Algebraic-geometry (AG) code. Parent of: Goppa code. Cousins: Evaluation AG code.
Alternant code[20] Stub. Parents: Linear \(q\)-ary code. Parent of: Generalized Reed-Solomon (GRS) code.
Group code An \( [n,k] \) binary or \(q\)-ary code based on a finite group \( G \) of size \(n \). A group code for an abelian group is called an abelian group code. Parents: Linear \(q\)-ary code. Parent of: Reed-Muller (RM) code. Cousins: Cyclic code.
Projective \(q\)-ary code Linear \([n,k,d]_q\) code such that columns of its generator matrix form points in the projective space \(PG(k-1,q)\). Parents: Linear \(q\)-ary code. Parent of: Simplex code.
Two-weight code Stub. Parents: Linear \(q\)-ary code.
Wozencraft ensemble code[21] Stub. Parents: Linear \(q\)-ary code. Cousin of: Justesen code.
Extended RS code Stub. If \(f\in \mathcal{P}_k\) with \(k<q\), then \(\sum_{\alpha\in\mathbb{F}_q}f(\alpha)=0\) which implies RS codes are odd-like. Hence, by adding a parity check coordinate with evaluation point \(\alpha_0=0\) to an RS code on \(q-1\) registers, the distance increases to \(\hat{d}=d+1\). This addition yields an \([q,k,q-k+1]\) extended RS code. Parents: Reed-Solomon (RS) code. Cousin of: Simplex code.
Folded RS code Stub. Parents: Reed-Solomon (RS) code.
\(q\)-ary quadratic-residue (QR) code Member of a quadruple of cyclic \(q\)-ary codes of prime length that satisfy certain relations ([1] Ch. 16). Stub. Parents: \(q\)-ary duadic code. Cousin of: Binary quadratic-residue (QR) code.
Simplex code An \([n,k,q^{k-1}]_q\) projective code with \(n=\frac{q^k-1}{q-1}\), denoted as \(S(k,q)\). The columns of the generator matrix are in one-to-one correspondence with the elements of the projective space \(PG(k-1,q)\), with each column being a chosen representative of the corresponding element. All of the non-zero codewords have a constant weight \(q^{k-1}\). Its dual code is the \((n,n-k,3)\) Hamming code. The name of the code comes from the property that, for \(q=2\), the codewords form a \((2^k-1)\)-simplex of constant edge length if the codewords are interpreted as points in \(\mathbb{R}^n\). Parents: Projective \(q\)-ary code. Cousins: Extended RS code, Binary repetition code, Binary Hamming code, Reed-Muller (RM) code.
