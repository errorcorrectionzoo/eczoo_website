Galois-field Kingdom

Welcome to the Galois-field Kingdom.

Galois-field \(q\)-ary code Encodes \(K\) states (codewords) in \(n\) \(q\)-ary coordinates over the field \(GF(q)=\mathbb{F}_q\) and has distance \(d\). Usually denoted as \((n,K,d)_q\). The distance is the minimum number of coordinates where two strings in the code differ. Protection: Detects errors on up to \(d-1\) coordinates, corrects erasure errors on up to \(d-1\) coordinates, and corrects general errors on up to \(\left\lfloor (d-1)/2 \right\rfloor\) coordinates. Parents: Error-correcting code. Parent of: Linear Galois-field \(q\)-ary code, Vector rank-metric code.
Linear Galois-field \(q\)-ary code An \((n,K,d)_q\) linear code is denoted as \([n,k,d]_q\), where \(k=\log_{q}K\) need not be an integer. Its codewords form a linear subspace, i.e., for any codewords \(x,y\), \(\alpha x+ \beta y\) is also a codeword for any \(q\)-ary digits \(\alpha,\beta\). Often defined in terms of a generator matrix \(G\), whose rows form a basis for the subspace. Given a codeword \(x\), the corresponding encoded codeword is \(G^T x\). Protection: Distance \(d\) of a linear code is the number of nonzero entries in the (nonzero) codeword with the smallest such number. Corrects any error set such that the difference of any pair of distinct elements of the set is a codeword. Parents: Galois-field \(q\)-ary code. Parent of: Alternant code, Dual linear code, Group code, Maximum distance separable (MDS) code, Quasi-cyclic code. Cousin of: Galois-qudit CSS code, Modular-qudit CSS code, Stabilizer code over \(GF(4)\), True Galois-qudit stabilizer code.
Vector rank-metric code[1][2] Also called Gabidulin or vector rank-metric code. The code corrects errors over rank metric instead of the traditional Hamming distance. The rank of an \(n\)-dimensional vector over \(GF(q)^N\) is the maximum number of its linearly independent coordinates. Protection: Set of vectors \(\{x_1, x_2, \ldots, x_M\}\) determines a rank code with distance \(d=\min d(x_i, x_j)\). The code with distance \(d\) corrects all errors with rank of the error not greater than \(\lfloor (d-1)/2\rfloor\). Parents: Galois-field \(q\)-ary code. Parent of: Maximum-rank vector rank-metric code.
Alternant code[3] Stub. Parents: Linear Galois-field \(q\)-ary code. Parent of: Bose–Chaudhuri–Hocquenghem (BCH) code, Generalized Reed-Solomon code, Goppa code.
Dual linear code[4] For any \([n,k]_q\) linear code \(C\), the dual code, \begin{align} C^\perp = \{ y\in GF(q)^{\times n} ~|~ x\cdot y=0 \forall x\in C\}, \end{align} is the row space of the parity check matrix of \(C\). The dual code is the kernel of the encoding map for \(C\), and \(\dim C^\perp = n-k\). Protection: The dual of an \([n,k,d] \) code is an \([n,n-k,d']\) code, where \(d'\) is not always related to \(d\). The generator matrix of \(C^\perp\) is the parity check matrix of \(C\), and visa versa. Parents: Linear Galois-field \(q\)-ary code. Cousin of: Calderbank-Shor-Steane (CSS) stabilizer code, Hadamard code, Majorana stabilizer code.
Group code An \( [n,k]_{q} \) code based on a finite group \( G \) of size \(n \). A group code for an abelian group is called an abelian group code. Parents: Linear Galois-field \(q\)-ary code. Cousins: Cyclic code, Reed-Muller code, Group-based quantum code.
Maximum distance separable (MDS) code[5] A \([n,k,d]_q\) \(q\)-ary linear code is an MDS code if parameters \(n\), \(k\), \(d\), and \(q\) are such that the Singleton bound \begin{align} d \leq n-k+1 \end{align} becomes an equality. A bound for general \(q\)-ary codes can also be formulated; see Thm. 1.9.10 in Ref. [4]. Protection: Given \(n\) and \(k\), MDS codes have the highest distance possible of all codes and so have the best possible error correction properties. Parents: Linear Galois-field \(q\)-ary code. Cousin of: Quantum maximum-distance-separable (MDS) code, Reed-Solomon code.
Quasi-cyclic code[6] Stub. Parents: Linear Galois-field \(q\)-ary code. Parent of: Skew-cyclic code.
Maximum-rank vector rank-metric code[1][2] Linear rank-metric code with maximum rank \([n,k,n-k+1]_q\) over rank metric. For any vector \(g=(g_1, \ldots, g_n)\) with linearly independent elements \(g_i\) over \(GF(q)\), the generator matrix \begin{align} G = \begin{pmatrix} g_1^{[0]} & g_2^{[0]} & \ldots & g_n^{[0]} \\ g_1^{[1]} & g_2^{[1]} & \ldots & g_n^{[1]} \\ g_1^{[k-1]} & g_2^{[k-1]} & \ldots & g_n^{[k-1]} \end{pmatrix}~, \end{align} where \([i]:=q^i\) and \(k=n-d+1\) defines a maximum rank distance code. Protection: \(d\)-distance codes protect agains all errors of rank not greater \(\lfloor (d-1)/2\rfloor\) Parents: Vector rank-metric code. Cousins: Reed-Solomon code.
Bose–Chaudhuri–Hocquenghem (BCH) code[7] Stub. Parents: Alternant code. Cousins: Reed-Solomon code, Cyclic code, Goppa code. Cousin of: Generalized Reed-Solomon code, Skew-cyclic code.
Generalized Reed-Solomon code[8] Stub. Parents: Alternant code. Parent of: Reed-Solomon code. Cousins: Goppa code, Bose–Chaudhuri–Hocquenghem (BCH) code. Cousin of: Quantum maximum-distance-separable (MDS) code.
Goppa code[9][10][11] Let \( G(z) \) be a polynomial describing a projective plane curve with coefficients from \( GF(q^m) \) for some fixed integer \(m\). Let \( L \) be a finite subset of the extension field \( GF(q^m) \) where \(q\) is prime, meaning \( L = \{\alpha_1, \cdots, \alpha_n\} \) is a subset of nonzero elements of \( GF(q^m) \). A Goppa code \( \Gamma(L,G) \) is an \([n,k,d]\) linear code consisting of all vectors \(a = a_1, \cdots, a_n\) such that \( R_a(z) =0 \) modulo \(G(z)\), where \( R_a(z) = \sum_{i=1}^n \frac{a_i}{z - \alpha_i} \). Protection: The length \( n = |L| \) , dimension \( k \geq n-mr \) where \( r = \text{deg} G(z) \), and the minimum distance \( d \geq r +1 \). Parents: Alternant code. Cousin of: Binary quantum Goppa code, Bose–Chaudhuri–Hocquenghem (BCH) code, Generalized Reed-Solomon code.
Skew-cyclic code[12] A classical code \(C\) of length \(n\) over \(GF(q)\) is skew-cyclic if there exists an automorphism, \(\theta\), of \(GF(q)\), such that for each string \(c_1 c_2 \cdots c_n\in C\), the skew-cyclically shifted string \(\theta(c_n) \theta(c_1) \cdots \theta(c_{n-1})\in C\). We say that \(C\) is a \(\theta\)-cyclic code over \(GF(q)\). Parents: Quasi-cyclic code. Parent of: Cyclic code. Cousins: Bose–Chaudhuri–Hocquenghem (BCH) code. Cousin of: Skew-cyclic CSS code.
Reed-Solomon code[13] An \([n,k,n-k+1]_q\) linear code based on polynomials over \(GF(q)\). Let \(\{\alpha_1,\cdots,\alpha_n\}\) be \(n\) distinct nonzero elements of \(GF(q)\) with \(q>n\). Encodes \(\mu=\{\mu_0,\cdots,\mu_{k-1}\}\) into \(\{f_\mu(\alpha_1),\cdots,f_\mu(\alpha_n)\}\), with polynomial \begin{align} f_\mu(x)=\mu_0+\mu_1 x + \cdots + \mu_{k-1}x^{k-1}. \end{align} Protection: Corrects erasures on \(n-k\) coordinates. Parents: Generalized Reed-Solomon code. Cousins: Hamming code, Maximum distance separable (MDS) code, Cyclic code. Cousin of: Approximate secret-sharing code, Bose–Chaudhuri–Hocquenghem (BCH) code, Convolutional code, Galois-qudit polynomial code (QPyC), Maximum-rank vector rank-metric code, Prime-qudit polynomial code (QPyC).
Cyclic code A classical code \(C\) of length \(n\) over an alphabet is cyclic if, for each string \(c_1 c_2 \cdots c_n\in C\), the cyclically shifted string \(c_n c_1 \cdots c_{n-1}\in C\). Parents: Skew-cyclic code. Cousin of: Bose–Chaudhuri–Hocquenghem (BCH) code, Galois-qudit polynomial code (QPyC), Group code, Majorana stabilizer code, Prime-qudit polynomial code (QPyC), Quantum cyclic code, Reed-Solomon code.
