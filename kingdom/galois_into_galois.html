Galois Field Kingdom

Welcome to the Galois Field Kingdom.

Galois-qudit code[1] Encodes \(K\)-dimensional Hilbert space into a \(q^n\)-dimensional (\(n\)-qudit) Hilbert space, with canonical qudit states \(|k\rangle\) labeled by elements \(k\) of the Galois field \(GF(q)\) and with \(q\) being a power of a prime \(p\). A Galois field can be thought of as a vector space whose basis vectors are the \(m\) roots of some polynomial and whose coefficients (i.e., field) are \(p\)th roots of unity. Codes can be denoted as \(((n,K))_{GF(q)}\) or \(((n,K,d))_{GF(q)}\), whenever the code's distance \(d\) is defined. Parents: Finite-dimensional QECC. Parent of: Galois-qudit stabilizer code. Cousins: \(\mathbb{Z}_q\)-qudit code.
Galois-qudit stabilizer code[2] An \(((n,p^k,d))_{GF(q)}\) stabilizer code is denoted as \([[n,k]]_{GF(q)}\) or \([[n,k,d]]_{GF(q)}\), where \(d\) is the code's distance. Stub. Parents: Galois-qudit code. Parent of: \(\mathbb{Z}_p\)-qudit stabilizer code, Galois-qudit CSS code.
\(\mathbb{Z}_p\)-qudit stabilizer code[3] An \(((n,p^k,d))_p\) (with \(p\) prime) stabilizer code is denoted as \([[n,k]]_p\) or \([[n,k,d]]_p\), where \(d\) is the code's distance. Logical subspace is the joint \(+1\) eigenspace of a set of \(p^{n-k}\) commuting qudit Pauli operators forming the code's stabilizer group, which cannot contain \(-I\). The distance is the minimum weight of an error that implements a nontrivial logical operation on the code. Normalizer \(N(S)\) of the stabilizer \(S\) is the set that contains all operators that commutes with \(S\). The stabilizer code can correct errors \({\mathcal{E}} \in P_n(p)\) if and only if \(\hat{E}^\dagger \hat{F} \notin \hat{N}(S)\setminus \hat{S}\) for all \(E,F \in {\mathcal{E}}\). Protection: Detects errors on up to \(d-1\) qudits, and corrects erasure errors on up to \(d-1\) qudits. Parents: \(\mathbb{Z}_q\)-qudit stabilizer code, Galois-qudit stabilizer code.
Galois-qudit CSS code Stub. Parents: Galois-qudit stabilizer code. Parent of: Galois-qudit polynomial code. Cousins: Calderbank-Shor-Steane (CSS) stabilizer code, \(\mathbb{Z}_q\)-qudit CSS code.
Galois-qudit polynomial code Stub. Parents: Galois-qudit CSS code.
