Welcome to the Ring Kingdom.

Ring code Encodes \(K\) states (codewords) in \(n\) coordinates over a ring \(R\). Parents: Error-correcting code (ECC). Parent of: \(R\)-linear code.
\(R\)-linear code A code of length \(n\) over a ring \(R\) is \(R\)-linear if it is a submodule of \(R^n\). Parents: Ring code. Parent of: Quaternary code over \(\mathbb{Z}_4\).
Reed-Muller (RM) code[1][2][3] Member of the RM\((r,m)\) family of linear binary codes derived from multivariate polynomials. The code parameters are \([2^m,\sum_{j=0}^{r} {m \choose j},2^{m-r}]\), where \(r\) is the order of the code satisfying \(0\leq r\leq m\). Punctured RM codes RM\(^*(r,m)\) are obtained from RM codes by deleting one or more coordinates from each codeword. Parents: Bose–Chaudhuri–Hocquenghem (BCH) code, Quaternary code over \(\mathbb{Z}_4\), Divisible code, Binary duadic code, Group code. Cousins: Dual linear code, Binary cyclic code. Cousin of: Hadamard code, Hamming code, Majorana stabilizer code, Polar code, Quantum Reed-Muller code, Quantum divisible code, Simplex code.
Quaternary code over \(\mathbb{Z}_4\) A linear code encoding \(K\) states (codewords) in \(n\) coordinates over the ring \(\mathbb{Z}_4\) of integers modulo 4. Parents: \(R\)-linear code. Parent of: Reed-Muller (RM) code.
