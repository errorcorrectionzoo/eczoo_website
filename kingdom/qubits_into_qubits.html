Qubit Kingdom

Welcome to the Qubit Kingdom.

Qubit code Encodes \(K\)-dimensional Hilbert space into a \(2^n\)-dimensional (i.e., \(n\)-qubit) Hilbert space. Usually denoted as \(((n,K))\) or \(((n,K,d))\), where \(d\) is the code's distance. Parent of: Stabilizer code.
Stabilizer code[1] An \(((n,2^k,d))\) stabilizer code is denoted as \([[n,k]]\) or \([[n,k,d]]\), where \(d\) is the code's distance. Logical subspace is the joint \(+1\) eigenspace of a set of \(2^{n-k}\) commuting Pauli operators forming the code's stabilizer group. The distance is the minimum weight of an error that implements a nontrivial logical operation on the code. Protection: Detects errors on up to \(d-1\) qubits, and corrects erasure errors on up to \(d-1\) qubits. Parents: Qubit code. Parent of: Calderbank-Shor-Steane (CSS) code, Quantum low-density parity-check (LDPC) code.
Calderbank-Shor-Steane (CSS) code[2][3][4] An \([[n,k,d]]\) stabilizer code whose encoding is based on two binary linear codes, an \([n,k_1,d_1]\) code \(C_1\) and \([n,k_2,d_2]\) code \(C_2\) with \(C_1^\perp \subseteq C_2\), such that \(k=k_1+k_2-n\) and \(d=\min\{d_1,d_2\}\). Protection: Detects errors on \(d-1\) qubits, corrects errors on \(\left\lfloor (d-1)/2 \right\rfloor\) qubits. Parents: Stabilizer code. Parent of: Kitaev's toric code, Homological code, Quantum repetition code. Cousins: Binary linear code.
Quantum low-density parity-check (LDPC) code Parents: Stabilizer code. Parent of: Kitaev's toric code.
Kitaev's toric code[5] A family of stabilizer codes whose generators are few-body \(X\)-type and \(Z\)-type Pauli strings associated to the stars and plaquettes, respectively, of a tessellation of a two-dimensional surface (with a qubit located at each edge of the tesselation). Toric often refers to the specific code construction on the two-dimensional torus, the construction on surfaces with boundaries is often called the planar code [6], and surface code is often an alternative name for the general construction. Protection: When defined on an \(L\times L\) square tiling of the torus, protects against \(L\) errors. More generally, the code distance is related to the homology of the tessellation [7]. Parents: Calderbank-Shor-Steane (CSS) code, Homological code, Quantum low-density parity-check (LDPC) code.
Homological code Add topological code entry as well. Parents: Calderbank-Shor-Steane (CSS) code. Parent of: Kitaev's toric code.
Quantum repetition code Encodes \(1\) qubit into \(n\) qubits according to \(|0\rangle\to|0\rangle^{\otimes n}\) and \(|1\rangle\to|1\rangle^{\otimes n}\). Protection: Detects bit-flip errors \(X\) on \(\left\lfloor (n-1)/2\right\rfloor\) qubits. Cannot detect any phase-flip errors \(Z\). Parents: Calderbank-Shor-Steane (CSS) code.
