Qubit Kingdom

Welcome to the Qubit Kingdom.

Qubit code Encodes \(K\)-dimensional Hilbert space into a \(2^n\)-dimensional (i.e., \(n\)-qubit) Hilbert space. Usually denoted as \(((n,K))\) or \(((n,K,d))\), where \(d\) is the code's distance. Parents: Finite-dimensional QECC. Parent of: Codeword stabilized (CWS) code. Cousin of: Fermionic code.
Calderbank-Shor-Steane (CSS) stabilizer code[1][2][3]

An \([[n,k,d]]\) stabilizer code whose stabilizer generators are either \(Z\)-type or \(X\)-type Pauli strings. The parity check matrix is of the form \begin{align} H=\begin{pmatrix}0 & H_{Z}\\ H_{X} & 0 \end{pmatrix}~. \label{eq:parity} \end{align}

Encoding is based on two binary linear codes, an \([n,k_X,d_X]\) code \(C_X\) and \([n,k_Z,d_Z]\) code \(C_Z\) with \(C_X^\perp \subseteq C_Z\), such that \(k=k_X+k_Z-n\) and \(d\geq\min\{d_X,d_Z\}\). The \(H_X\) (\(H_Z\)) block of \(H\) \eqref{eq:parity} is associated with the code \(C_X\) (\(C_Z\)), and the requirement \(C_X^\perp \subseteq C_Z\) guarantees that the \(X\)-stabilizers of \(C_X\) commute with the \(Z\)-stabilizers of \(C_Z\).

A CSS code has stabilizer weight \(w\) if the highest weight of any stabilizer generator is \(w\), i.e., any row and column of \(H_X\) and \(H_Z\) has weight at most \(w\). In the context of comparing weight as well as of noise models biased toward \(X\)- or \(Z\)-type errors, an extended notation for CSS codes is \([[n,k,(d_X,d_Z),w]]\).

 Protection: Detects errors on \(d-1\) qubits, corrects errors on \(\left\lfloor (d-1)/2 \right\rfloor\) qubits. Parents: Stabilizer code, Movassagh-Ouyang Hamiltonian Codes. Parent of: Qubit repetition code, Homological product code, Kitaev's surface code, Quantum polar code. Cousins: Binary linear code, Weakly self-dual linear code. Cousin of: Majorana stabilizer code, Group GKP code, \(\mathbb{Z}_q\)-qudit CSS code, Galois-qudit CSS code.
Kitaev's surface code[4][5] A family of CSS stabilizer codes whose generators are few-body \(X\)-type and \(Z\)-type Pauli strings associated to the stars and plaquettes, respectively, of a tessellation of a two-dimensional surface (with a qubit located at each edge of the tesselation). Toric code often either refers to the construction on the two-dimensional torus or is an alternative name for the general construction. The construction on surfaces with boundaries is often called the planar code [6]. Protection: When defined on an \(L\times L\) square tiling of the torus, protects against \(L\) errors. More generally, the code distance is related to the homology of the tessellation [7]. Parents: Calderbank-Shor-Steane (CSS) stabilizer code, Topological code. Cousins: Quantum low-density parity-check (QLDPC) code, Hypergraph product code.

Majorana stabilizer code[8] Majorana fermion stabilizer codes are stabilizer codes whose stabilizers are products of an even number of Majorana fermion operators, analogous to Pauli strings for a traditional stabilizer code and referred to as Majorana stabilizers. The codespace is the mutual \(+1\) eigenspace of all Majorana stabilizers. In such systems, Majorana fermions may either be considered individually or paired into creation and annihilation operators for fermionic modes. Codes can be denoted as \([[n,k,d]]_{f}\) [9], where \(n\) is the number of fermionic modes. Protection: Detects products of Majorana operators with weight up to \(d-1\). Physically, protects against dephasing errors caused by coupling of fermion density to the environment and bit-flip errors caused by quasiparticle poisoning processes. Parents: Fermionic code, Stabilizer code. Cousins: Weakly self-dual linear code, Calderbank-Shor-Steane (CSS) stabilizer code, Cyclic code, Reed-Muller code.
Homological product code[10] Stub. Parents: Calderbank-Shor-Steane (CSS) stabilizer code, Quantum low-density parity-check (QLDPC) code. Parent of: Lifted product code, Hypergraph product code.
Qubit repetition code Encodes \(1\) qubit into \(n\) qubits according to \(|0\rangle\to|\phi_0\rangle^{\otimes n}\) and \(|1\rangle\to|\phi_1\rangle^{\otimes n}\). Also known as a bit-flip code when \(|\phi_i\rangle = |i\rangle\), and a phase-flip code when \(|\phi_0\rangle = |+\rangle\) and \(|\phi_1\rangle = |-\rangle\). Protection: Bit-flip code detects bit-flip errors \(X\) on \(\left\lfloor (n-1)/2\right\rfloor\) qubits and does not detect any phase-flip errors \(Z\). Phase-flip code detects phase-flip errors \(Z\) on \(\left\lfloor (n-1)/2\right\rfloor\) qubits and does not detect any bit-flip errors \(X\). Parents: Calderbank-Shor-Steane (CSS) stabilizer code. Cousins: Hamiltonian-based code.
Quantum polar code[11] Stub. Parents: Calderbank-Shor-Steane (CSS) stabilizer code. Cousins: Polar code.
Stabilizer code[12][13] An \(((n,2^k,d))\) stabilizer code is denoted as \([[n,k]]\) or \([[n,k,d]]\), where \(d\) is the code's distance. Logical subspace is the joint \(+1\) eigenspace of a set of \(2^{n-k}\) commuting Pauli operators forming the code's stabilizer group, which cannot contain \(-I\). The distance is the minimum weight of a Pauli string that implements a nontrivial logical operation on the code. Protection: Detects errors on up to \(d-1\) qubits, and corrects erasure errors on up to \(d-1\) qubits. Parents: Codeword stabilized (CWS) code. Parent of: Majorana stabilizer code, Quantum low-density parity-check (QLDPC) code, Quantum Hamming code, Calderbank-Shor-Steane (CSS) stabilizer code, Quantum Reed-Muller code. Cousins: Hamiltonian-based code. Cousin of: Gottesman-Kitaev-Preskill (GKP) code.
Lifted product code[14] A family of \([[n,k,d]]\) CSS codes whose construction is based on two linear seed codes \(C_1\) and \(C_2\) with a free action of a group \(G\). Lifted product codes can also be defined over arbitrary matrix rings. Protection: Code performance strongly depends on \(G\), but lifted product codes contain [15] families of QLDPC codes with the parameters \([[n, k = \Theta(n), d = \Theta(n)]]\). Parents: Homological product code. Parent of: Hypergraph product code.
Hypergraph product code[16][17] A family of \([[n,k,d]]\) CSS codes whose construction is based on two binary linear seed codes \(C_1\) and \(C_2\). Protection: The hypergraph product has distance \(d=O(\sqrt{n})\). The number of encoded logical qubits is \(k=O(k_1k_2)\) where \(k_1\) and \(k_2\) are the dimensions of the classical seed codes \(C_1\) and \(C_2\). Parents: Lifted product code, Homological product code. Cousin of: Kitaev's surface code.
Quantum low-density parity-check (QLDPC) code Parents: Stabilizer code. Parent of: Homological product code. Cousins: Low-density parity-check (LDPC) code, Hamiltonian-based code, Dynamically-generated quantum error-correcting code. Cousin of: Kitaev's surface code, Topological code.
