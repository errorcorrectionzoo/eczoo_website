Qubit Kingdom

Welcome to the Qubit Kingdom.

Qubit code Encodes \(K\)-dimensional Hilbert space into a \(2^n\)-dimensional (i.e., \(n\)-qubit) Hilbert space. Usually denoted as \(((n,K))\) or \(((n,K,d))\), where \(d\) is the code's distance. Parents: Quantum error-correcting code (QECC) in finite dimensions. Parent of: Stabilizer code.
Kitaev's surface code[1][2] A family of stabilizer codes whose generators are few-body \(X\)-type and \(Z\)-type Pauli strings associated to the stars and plaquettes, respectively, of a tessellation of a two-dimensional surface (with a qubit located at each edge of the tesselation). Toric code often either refers to the construction on the two-dimensional torus or is an alternative name for the general construction. The construction on surfaces with boundaries is often called the planar code [3]. Protection: When defined on an \(L\times L\) square tiling of the torus, protects against \(L\) errors. More generally, the code distance is related to the homology of the tessellation [4]. Parents: Calderbank-Shor-Steane (CSS) stabilizer code, Homological code, Quantum low-density parity-check (QLDPC) code, Topological code. Cousins: Hypergraph product code, Hamiltonian-based code.
Stabilizer code[5] An \(((n,2^k,d))\) stabilizer code is denoted as \([[n,k]]\) or \([[n,k,d]]\), where \(d\) is the code's distance. Logical subspace is the joint \(+1\) eigenspace of a set of \(2^{n-k}\) commuting Pauli operators forming the code's stabilizer group. The distance is the minimum weight of a Pauli string that implements a nontrivial logical operation on the code. Protection: Detects errors on up to \(d-1\) qubits, and corrects erasure errors on up to \(d-1\) qubits. Parents: Qubit code. Parent of: Calderbank-Shor-Steane (CSS) stabilizer code, Quantum Hamming code, Quantum low-density parity-check (QLDPC) code. Cousins: Hamiltonian-based code. Cousin of: Gotteman-Kitaev-Preskill (GKP) code.
Calderbank-Shor-Steane (CSS) stabilizer code[6][7][8] An \([[n,k,d]]\) stabilizer code whose encoding is based on two binary linear codes, an \([n,k_1,d_1]\) code \(C_1\) and \([n,k_2,d_2]\) code \(C_2\) with \(C_1^\perp \subseteq C_2\), such that \(k=k_1+k_2-n\) and \(d\geq\min\{d_1,d_2\}\). Protection: Detects errors on \(d-1\) qubits, corrects errors on \(\left\lfloor (d-1)/2 \right\rfloor\) qubits. Parents: Stabilizer code. Parent of: Homological code, Homological product code, Hypergraph product code, Lifted product code, Qubit repetition code, Kitaev's surface code. Cousins: Binary linear code. Cousin of: Group GKP codes.
Quantum Hamming code[9] Stub. Parents: Stabilizer code.
Quantum low-density parity-check (QLDPC) code Parents: Stabilizer code. Parent of: Homological product code, Hypergraph product code, Lifted product code, Kitaev's surface code. Cousins: Hamiltonian-based code, Dynamically-generated error-correcting code. Cousin of: Topological code.
Homological code Stub. Parents: Calderbank-Shor-Steane (CSS) stabilizer code. Parent of: Hypergraph product code, Kitaev's surface code.
Homological product code[10] Stub. Parents: Calderbank-Shor-Steane (CSS) stabilizer code, Quantum low-density parity-check (QLDPC) code. Parent of: Lifted product code.
Hypergraph product code[11] A family of \([[n,k,d]]\) CSS codes whose construction is based on two binary linear seed codes \(C_1\) and \(C_2\). Protection: The hypergraph product has distance \(d=O(\sqrt{n})\). The number of encoded logical qubits is \(k=O(k_1k_2)\) where \(k_1\) and \(k_2\) are the dimensions of the classical seed codes \(C_1\) and \(C_2\). Parents: Calderbank-Shor-Steane (CSS) stabilizer code, Homological code, Quantum low-density parity-check (QLDPC) code. Cousin of: Lifted product code, Kitaev's surface code.
Lifted product code[12] A family of \([[n,k,d]]\) CSS codes whose construction is based on two linear seed codes \(C_1\) and \(C_2\) with a free action of a group G. Protection: The lifted product codes contain [13] families of QLDPC codes with the parameters \([[n, k = \Theta(n), d = \Theta(n)]]\). Parents: Calderbank-Shor-Steane (CSS) stabilizer code, Homological product code, Quantum low-density parity-check (QLDPC) code. Cousins: Hypergraph product code.
Qubit repetition code Encodes \(1\) qubit into \(n\) qubits according to \(|0\rangle\to|\phi_0\rangle^{\otimes n}\) and \(|1\rangle\to|\phi_1\rangle^{\otimes n}\). Also known as a bit-flip code when \(|\phi_i\rangle = |i\rangle\), and a phase-flip code when \(|\phi_0\rangle = |+\rangle\) and \(|\phi_1\rangle = |-\rangle\). Protection: Bit-flip code detects bit-flip errors \(X\) on \(\left\lfloor (n-1)/2\right\rfloor\) qubits and does not detect any phase-flip errors \(Z\). Phase-flip code detects phase-flip errors \(Z\) on \(\left\lfloor (n-1)/2\right\rfloor\) qubits and does not detect any bit-flip errors \(X\). Parents: Calderbank-Shor-Steane (CSS) stabilizer code. Cousins: Hamiltonian-based code.
