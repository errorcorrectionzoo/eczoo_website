Stabilizer code

Name
Stabilizer code §
Description
An \(n,2^k,d\) stabilizer is usually denoted as \([[n,k,d]]\). Logical subspace is the joint \(+1\) eigenspace of a set of commuting Pauli operators forming the code's stabilizer group.
Protection
Parent(s)

Parent of
  • Calderbank-Shor-Steane (CSS) code (Stabilizer generators can be expressed as either only-\(X\) or only-\(Z\).)
  • Kitaev's toric code (Plaquette (\(Z\)) and star (\(X\)) operators are stabilizer generators.)
  • Quantum repetition code (Stabilizer generators can be expressed as either only-\(X\) or only-\(Z\).)
