Error correction zoo: qubits into qubits

This is a comprehensive catalog of classical and quantum error correcting codes. If you notice any errors or omissions, please email me at vva@umd.edu. (Alternatively, you may submit a pull request to the repository on github.) Your help is appreciated and will be acknowledged. The zoo is styled similar in spirit to the algorithm zoo.

Name: Repetition code
Encoding: \(1\) qubit into \(n\) qubits according to \(|0\rangle\to|0\rangle^{\otimes n}\) and \(|1\rangle\to|1\rangle^{\otimes n}\).
Protection: bit-flip errors \(X\) on \(\left\lfloor (n-1)/2\right\rfloor\) qubits
Features: Logical qubit spanned by GHz states. Extremely sensitive to phase-flip errors \(Z\).
Example of: CSS code, Stabilizer code (with stabilizers of the form \(Z\otimes Z\)).

Name: CSS code
Encoding: \([[n,k,d]]\) code based on two classical linear codes: an \([n,k_1,d_1]\) code \(C_1\) and \([n,k_2,d_2]\) code \(C_2\) such that \(C_1^\perp \subseteq C_1\). Then, \(k=k_1+k_2-n\) and \(d=\min\{d_1,d_2\}\).
Protection: detects errors on \(d-1\) qubits, corrects errors on \((d-1)/2\) qubits
Features:
Example of: Stabilizer code
Special cases: Repetition code

Name: Stabilizer code
Encoding: \([[n,k,d]]\) code whose logical subspace is the joint \(+1\) eigenspace of a set of commuting Pauli strings forming the code's stabilizer group.
Protection: detects errors on \(d-1\) qubits, corrects errors on \((d-1)/2\) qubits
