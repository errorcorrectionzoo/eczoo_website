Encoding qubits into qubits

Qubit error-correcting code Encodes \(K\)-dimensional Hilbert space into a \(2^n\)-dimensional (\(n\)-qubit) Hilbert space; has distance \(d\). Usually denoted as \(((n,K,d))\), or \([[n,k,d]]\) when \(K=2^k\). The distance is the minimum weight of an error that implements a nontrivial logical operation on the code. Detects errors on up to \(d-1\) qubits, and corrects erasure errors on up to \(d-1\) qubits. Parent of: Calderbank-Shor-Steane (CSS) code, Kitaev's toric code, Stabilizer code, Quantum repetition code, Quantum low-density parity-check (LDPC) code
Calderbank-Shor-Steane (CSS) code An \((n,2^k,d)\) CSS code is usually denoted as \([[n,k,d]]\). Encoding is based on two binary linear codes, an \([n,k_1,d_1]\) code \(C_1\) and \([n,k_2,d_2]\) code \(C_2\) with \(C_1^\perp \subseteq C_2\), such that \(k=k_1+k_2-n\) and \(d=\min\{d_1,d_2\}\). Detects errors on \(d-1\) qubits, corrects errors on \(\left\lfloor (d-1)/2 \right\rfloor\) qubits. Parents: Qubit error-correcting code, Stabilizer code Parent of: Kitaev's toric code Cousins: Binary linear code
Kitaev's toric code A family of stabilizer codes whose generators are few-body \(X\)-type Pauli strings (stars) and few-body \(Z\)-type Pauli strings on any surface plaquette operators on an arbitrary lattice on an arbitrary orientable 2D surface [1]. Parents: Qubit error-correcting code, Calderbank-Shor-Steane (CSS) code, Stabilizer code
Stabilizer code An \(n,2^k,d\) stabilizer is usually denoted as \([[n,k,d]]\). Logical subspace is the joint \(+1\) eigenspace of a set of commuting Pauli operators forming the code's stabilizer group. Parents: Qubit error-correcting code Parent of: Calderbank-Shor-Steane (CSS) code, Kitaev's toric code, Quantum repetition code
Quantum repetition code Encodes \(1\) qubit into \(n\) qubits according to \(|0\rangle\to|0\rangle^{\otimes n}\) and \(|1\rangle\to|1\rangle^{\otimes n}\). Detects bit-flip errors \(X\) on \(\left\lfloor (n-1)/2\right\rfloor\) qubits. Cannot detect any phase-flip errors \(Z\). Parents: Qubit error-correcting code, Stabilizer code Cousins: Binary linear code
Quantum low-density parity-check (LDPC) code Detects errors on \(d-1\) qubits, corrects errors on \(\left\lfloor (d-1)/2 \right\rfloor\) qubits. Parents: Qubit error-correcting code
