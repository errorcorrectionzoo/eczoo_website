Index of all codes

Qudit-into-oscillator code Encodes \(K\)-dimensional Hilbert space into Hilbert space of \(\ell^2\)-normalizable functions on \(\mathbb{R}^n\).
Oscillator code Encodes Hilbert space of \(\ell^2\)-normalizable functions on \(\mathbb{R}^k\) into that on \(\mathbb{R}^n\). Usually denoted as \(((n,k))_{\mathbb{R}}\).
Qubit code Encodes \(K\)-dimensional Hilbert space into a \(2^n\)-dimensional (\(n\)-qubit) Hilbert space; has distance \(d\). Usually denoted as \(((n,K,d))\), or \([[n,k,d]]\) when \(K=2^k\). The distance is the minimum weight of an error that implements a nontrivial logical operation on the code. Detects errors on up to \(d-1\) qubits, and corrects erasure errors on up to \(d-1\) qubits. Parent of: Calderbank-Shor-Steane (CSS) code, Kitaev's toric code, Stabilizer code, Quantum repetition code, Quantum low-density parity-check (LDPC) code
Group code Encodes Hilbert space of \(\ell^2\)-normalizable functions on a group \(G_1\) into that on a second-countable unimodular group \(G_2\). For finite \(G_1\) and \(G_2=G^{\times n}\), can be denoted as \(((n,|G_1|))_G\). Furthermore, when \(K=|G|^k\), can be denoted \([[n,k]]_G\). Ideal codewords may not be normalizable, depending on whether \(G_2\) is continuous and/or noncompact, so approximate versions have to be constructed. Parent of: Group GKP codes
Qudit code Encodes \(K\)-dimensional Hilbert space into a \(q^n\)-dimensional (\(n\)-qudit) Hilbert space for some \(q\); has distance \(d\). Usually denoted as \(((n,K,d))_q\), or \([[n,k,d]]_q\) when \(K=q^k\). The distance is the minimum weight of an error that implements a nontrivial logical operation on the code. Detects errors on up to \(d-1\) qubits, and corrects erasure errors on up to \(d-1\) qubits.
Binary code Encodes \(K\) bitstrings (codewords) into bitstrings of length \(n\) and has distance \(d\). Usually denoted as \((n,K,d)\), or \([n,k,d]\) when \(K=2^k\). The distance is the number of nonzero entries in the (nonzero) codeword with the smallest such number. Detects errors on up to \(d-1\) coordinates, corrects erasure errors on up to \(d-1\) coordinates, and corrects general errors on up to \(\left\lfloor (d-1)/2 \right\rfloor\) coordinates. Parent of: Binary linear code
q-ary code Encodes \(K\) \(q\)-ary strings (codewords) into \(q\)-ary strings of length \(n\) and has distance \(d\). Usually denoted as \((n,K,d)_q\), or \([n,k,d]_q\) when \(K=q^k\). The distance is the number of nonzero entries in the (nonzero) codeword with the smallest such number. Detects errors on up to \(d-1\) coordinates, corrects erasure errors on up to \(d-1\) coordinates, and corrects general errors on up to \(\left\lfloor (d-1)/2 \right\rfloor\) coordinates. Parent of: Reed-Solomon code
Stabilizer code An \(n,2^k,d\) stabilizer is usually denoted as \([[n,k,d]]\). Logical subspace is the joint \(+1\) eigenspace of a set of commuting Pauli operators forming the code's stabilizer group. Parents: Qubit code Parent of: Calderbank-Shor-Steane (CSS) code, Kitaev's toric code, Quantum repetition code
Quantum low-density parity-check (LDPC) code Detects errors on \(d-1\) qubits, corrects errors on \(\left\lfloor (d-1)/2 \right\rfloor\) qubits. Parents: Qubit code
Group GKP codes Encodes finite-dimensional Hilbert space into the Hilbert space of \(\ell^2\)-normalizable functions on a group \(G\). Construction is based on nested subgroups \(H\subset K \subset G\), where \(H,K\) are finite. The \(|K|/|H|\)-dimensional logical subspace is spanned by basis states that are equal superpositions of elements of cosets of \(H\) in \(K\). Protects against generalized bit-flip errors \(g\in G\) that are inside the fundamental domain of \(G/K\). Protection against phase-flip errors determined by branching rules of irreps of \(G\) into those of \(K\), and further into those of \(H\). Parents: Group code Parent of: Molecular codes
Binary linear code An \((n,2^k,d)\) linear code is denoted as \([n,k,d]\). Its codewords form a linear subspace, i.e., for any codewords \(x,y\), \(x+y\) is also a codeword. Often defined in terms of a generator matrix \(G\), whose rows form a basis for the subspace. Given a codeword labeled by a binary string \(v\), the corresponding encoded codeword is \(G^T v\). A classical code with distance \(d\) detects errors on up to \(d-1\) coordinates, and corrects general errors on up to \(\left\lfloor (d-1)/2 \right\rfloor\) coordinates. Distance \(d\) of a linear code is the number of nonzero entries in the (nonzero) codeword with the smallest such number. Generally, linear codes correct any error set such that no two elements of the set add up to a codeword. Parents: Binary code Cousin of: Calderbank-Shor-Steane (CSS) code, Quantum repetition code
Reed-Solomon code \([n,k,n-k+1]_q\) linear code. Let \(\{\alpha_1,\cdots,\alpha_n\}\) be \(n\) distinct nonzero elements of \(GF(q)\) for prime \(q>n\). Encodes \(\mu=\{\mu_0,\cdots,\mu_{k-1}\}\) into \(\{f_\mu(\alpha_1),\cdots,f_\mu(\alpha_n)\}\) with polynomial \(f_\mu(x)=\mu_0+\mu_1 x + \cdots + \mu_{k-1}x^{k-1}\). Corrects erasures on \(n-k\) coordinates. Parents: q-ary code
Calderbank-Shor-Steane (CSS) code An \((n,2^k,d)\) CSS code is usually denoted as \([[n,k,d]]\). Encoding is based on two binary linear codes, an \([n,k_1,d_1]\) code \(C_1\) and \([n,k_2,d_2]\) code \(C_2\) with \(C_1^\perp \subseteq C_2\), such that \(k=k_1+k_2-n\) and \(d=\min\{d_1,d_2\}\). Detects errors on \(d-1\) qubits, corrects errors on \(\left\lfloor (d-1)/2 \right\rfloor\) qubits. Parents: Qubit code, Stabilizer code Parent of: Kitaev's toric code Cousins: Binary linear code
Quantum repetition code Encodes \(1\) qubit into \(n\) qubits according to \(|0\rangle\to|0\rangle^{\otimes n}\) and \(|1\rangle\to|1\rangle^{\otimes n}\). Detects bit-flip errors \(X\) on \(\left\lfloor (n-1)/2\right\rfloor\) qubits. Cannot detect any phase-flip errors \(Z\). Parents: Qubit code, Stabilizer code Cousins: Binary linear code
Molecular codes Encodes finite-dimensional Hilbert space into the Hilbert space of \(\ell^2\)-normalizable functions on the group \(SO_3\). Construction is based on nested subgroups \(H\subset K \subset SO_3\), where \(H,K\) are finite. The \(|K|/|H|\)-dimensional logical subspace is spanned by basis states that are equal superpositions of elements of cosets of \(H\) in \(K\). Protects against generalized bit-flip errors \(g\in SO_3\) that are inside the fundamental domain of \(G/K\). Protection against phase-flip errors determined by branching rules of irreps of \(G\) into those of \(K\), and further into those of \(H\). Parents: Group GKP codes
Kitaev's toric code A family of stabilizer codes whose generators are few-body \(X\)-type Pauli strings (stars) and few-body \(Z\)-type Pauli strings on any surface plaquette operators on an arbitrary lattice on an arbitrary orientable 2D surface [1]. Parents: Qubit code, Calderbank-Shor-Steane (CSS) code, Stabilizer code
