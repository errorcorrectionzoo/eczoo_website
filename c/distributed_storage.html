Distributed-storage code

Description

Designed to encode information into spatial nodes such that it is possible to recover said information after failure of some nodes by accessing the remaining nodes with minimal bandwidth. Stub.

Parent

  • Error-correcting code (ECC)

Children

  • Generalized Reed-Solomon code — Generalized RS codes are used in various cloud storage systems [1].
  • Locally recoverable code (LRC)
  • Maximum distance separable (MDS) code — MDS codes are most efficient in terms of minimizing storage overhead for handling erasures.
  • Regenerating code (RGC)
