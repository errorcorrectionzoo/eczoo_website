Matching code[1]

Description

Stub.

Realizations

Braiding of defects has been demonstrated for a five-qubit version of code [2].

Parents

  • Qubit stabilizer code
  • Abelian topological code — Matching codes were inspired by the Kitaev honeycomb model [3], which realizes \(\mathbb{Z}_2\) topological order.

Child

  • XYZ\(^2\) hexagonal stabilizer code

Cousin

  • Honeycomb Floquet code — Matching and honeycomb codes are both inspired by the Kitaev honeycomb model [3].

References
