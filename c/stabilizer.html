Stabilizer code

Name
Stabilizer code      
Description
An \(n,2^k,d\) qubit code is usually denoted as \([[n,k,d]]\). Logical subspace is the joint \(+1\) eigenspace of a set of \(2^{n-k}\) commuting Pauli operators forming the code's stabilizer group.
