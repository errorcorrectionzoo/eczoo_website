Klein-quartic code[1]

Description

Evaluation AG code over \(GF(8)\) of rational functions evaluated on points lying in the Klein quartic, which is defined by the equation \(x^3 y + y^3 z + z^3 x = 0\) ([2], Ex. 2.75).

Protection

Dimension \(k=8\) and distance \(d \geq 13\). Concatenation with the \([4,3,2]\) single parity check code, conversion to a binary code by expressing \(GF(8)\) elements as vectors over \(GF(2)\), and puncturing yields a \([91,24,25]\) binary code that held the world record for codes of length 91 [3].

Parent

  • Evaluation AG code — Klein-quartic codes are evaluation AG codes with \(\cal X\) being the Klein quartic ([2], Ex. 2.75).
