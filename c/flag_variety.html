Flag-variety code[1]

Description

Evaluation code of polynomials evaluated on points lying on a flag variety.

Parent

  • Polynomial evaluation code — Flag-variety codes are polynomial evaluation codes with \(\cal X\) being a flag variety.

Children

  • Grassmannian code — Grassmannian codes are flag-variety evaluation codes with the flag variety being a Grassmannian.
  • Hermitian-hypersurface code — Hermitian-hypersurface codes are flag-variety evaluation codes with the flag variety being a Hermitian hypersurface.
  • Quadric code — Quadric codes are flag-variety evaluation codes with the flag variety being a quadric hypersurface.
  • Schubert code — Schubert codes are flag-variety evaluation codes with the flag variety being a Schubert variety.

References

[1]
F. Rodier, “Codes from flag varieties over a finite field”, Journal of Pure and Applied Algebra 178, 203 (2003). DOI
