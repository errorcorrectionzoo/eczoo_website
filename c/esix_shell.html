\(E_6\) lattice-shell code

Description

Spherical code whose codewords are points on the \(E_6\) lattice normalized to lie on the unit sphere.

The smallest shell of the lattice yields the \((6,72,1)\) code, whose codewords form the vertices of the \(1_{22}\) real polytope and the rectified Hessian polyhedron.
