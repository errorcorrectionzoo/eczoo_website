Dinur-Hsieh-Lin-Vidick (DHLV) code[1]

Description

Stub.

Rate

Asymptotically good QLDPC codes.

Decoding

Linear-time decoder utilizing the small set-flip decoder [2] for \(Z\) errors and a reconstruction procedure for \(X\) errors [1].

Parent

  • Balanced product code

Cousins

  • Good QLDPC code — DHLV code construction yields asymptotically good QLDPC codes.
  • Tanner code — Tanner codes are used in constructing quantum DHLV codes.
  • Tensor-product code — Tensor codes are used in constructing quantum DHLV codes.

References

Irit Dinur et al., “Good Quantum LDPC Codes with Linear Time Decoders”. 2206.07750
[2]
Shouzhen Gu, Christopher A. Pattison, and Eugene Tang, “An efficient decoder for a linear distance quantum LDPC code”. 2206.06557
