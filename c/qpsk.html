Quadrature PSK (QPSK) code[1]

Description

Also known as quadriphase PSK, 4-PSK, or 4-QAM. Quaternary encoding into a constellation of four points distributed equidistantly on a circle. For the case of \(\pi/4\)-QPSK, the constellation is \(\{e^{\pm i\frac{\pi}{4}},e^{\pm i\frac{3\pi}{4}}\}\).

Realizations

Japanese and North American digital cellular and personal systems [2].Telephone-line modems: 1962 Bell 201 and international standard V.24 [3].

Parents

  • Phase-shift keyring (PSK) code — QPSK is also known as 4-PSK.
  • Biorthogonal spherical code — QPSK code is equivalent to the biorthogonal spherical code for \(n=2\).
