Generalized Reed Solomon (GRS) code[1]

Description

An \([n,k,n-k+1]_q\) linear code based on polynomials over \(GF(q)\) multiplied by additional prefactors. Each codeword \(\mu\) of a Reed-Solomon (RS) code is a string of values of a corresponding polynomial \(f_\mu\) at the points \(\alpha_i\). Codewords of a GRS code multiply the polynomial by a factor \(v_i\) for each \(i\). In other words, each codeword becomes \( ( v_1 f_{\mu}(\alpha_1), v_2 f_{\mu}(\alpha_2), \cdots, v_n f_{\mu}(\alpha_n) ) \).

The decoding process of GRS codes reduces to the solution of a polynomial congruence equation, usually referred to as the key equation. Decoding schemes are based on applications of Euclid algorithm to solve the key equation.

Protection

The code can detect \(n-k\) errors, and can correct errors \( \left\lfloor (n-k)/2\right\rfloor \) errors.

Decoding

Berlekamp-Massey decoder [2][3][4].Guruswami-Sudan list decoder [5] and modification by Koetter-Vardy for soft-decision decoding [6].

Realizations

Various cloud storage systems [7].Public-key cryptosystems [8]. Initial construction of McEliece Public Key Cryptosystem was based on Goppa codes which are subfield subcode of GRS codes. Public Key Cryptosystem designs based on GRS codes first proposed in Ref. [9], which replaced the generator matrix with the parity check matrix, were proven to be insecure [10] since the public key exposes algebraic structure of code. More recent works focus on methods to mask the algebraic structure using subcodes of GRS codes [8]. Lately a key-recovery attack was developed in [11] for a variant of masking method proposed by [12].

Parents

  • Evaluation AG code — GRS codes are evaluation AG codes on the projetive line; see Thm. 15.3.24 of Ref. [13].
  • Distributed-storage code — GRS codes are used in various cloud storage systems [7].
  • Maximum distance separable (MDS) code — GRS codes have distance \(n-k+1\), saturating the Singleton bound.

Children

  • Alternant code — Alternant codes are subfield subcodes of GRS codes.
  • Bose–Chaudhuri–Hocquenghem (BCH) code — BCH codes are subfield subcodes of GRS codes.
  • Goppa code — Goppa codes are subfield subcodes of GRS codes.
  • Reed-Solomon (RS) code — A GRS code for whic all mujltipliers \(v_i=1\) reduces to an RS code.

Cousin

  • Quantum maximum-distance-separable (MDS) code — Quantum MDS codes can be constructed through classical generalized Reed-Solomon codes [14].
