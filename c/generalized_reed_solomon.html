Generalized Reed Solomon (GRS) code[1]

Description

An \([n,k,n-k+1]_q\) linear code that is a modification of the RS code where codeword polynomials are multiplied by additional prefactors. Each codeword \(\mu\) is a string of values of the corresponding polynomial \(f_\mu\) at the points \(\alpha_i\), multiplied by a corresponding factor \(v_i \in GF(q)\), \begin{align} \mu\to\left\{ v_{1}f_{\mu}\left(\alpha_{1}\right),v_{2}f_{\mu}\left(\alpha_{2}\right),\cdots,v_{n}f_{\mu}\left(\alpha_{n}\right)\right\}~. \end{align}

Protection

The code can detect \(n-k\) errors, and can correct errors \( \left\lfloor (n-k)/2\right\rfloor \) errors.

Decoding

The decoding process of GRS codes reduces to the solution of a polynomial congruence equation, usually referred to as the key equation. Decoding schemes are based on applications of the Euclid algorithm to solve the key equation.Berlekamp-Massey decoder with runtime of order \(O(n^2)\) [2][3][4].Guruswami-Sudan list decoder [5] and modification by Koetter-Vardy for soft-decision decoding [6].

Realizations

Various cloud storage systems [7].Public-key cryptosystems generalizing those that used Goppa codes [8][9][10], some of which were proven to be insecure [11]. More recent works focus on methods to mask the algebraic structure using subcodes of GRS codes [12]. For example, a key-recovery attack was developed in Ref. [13] for a variant of masking method proposed in Ref. [14].

Parents

  • Linear \(q\)-ary code
  • Evaluation AG code — GRS codes are evaluation AG codes on the projetive line; see Thm. 15.3.24 of Ref. [15].
  • Distributed-storage code — GRS codes are used in various cloud storage systems [7].

Children

  • Alternant code — Alternant codes are subfield subcodes of GRS codes.
  • Bose–Chaudhuri–Hocquenghem (BCH) code — BCH codes are subfield subcodes of GRS codes.
  • Goppa code — Goppa codes are subfield subcodes of GRS codes.
  • Reed-Solomon (RS) code — A GRS code for which all multipliers \(v_i\) are unity reduces to an RS code.

Cousins

  • Maximum distance separable (MDS) code — GRS codes have distance \(n-k+1\), saturating the Singleton bound.
  • Quantum maximum-distance-separable (MDS) code — Quantum MDS codes can be constructed through classical generalized Reed-Solomon codes [16].
