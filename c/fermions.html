Fermionic code

Description

Finite-dimensional quantum error-correcting code encoding a logical Hilbert space into a physical Fock space of fermionic modes. Codes are typically described using Majorana operators, which are linear combinations of fermionic creation and annihilation operators [1].

Parent

  • Finite-dimensional quantum error-correcting code

Child

  • Majorana stabilizer code

Cousins

  • Qubit code — While the Majorana operator algebra is isomorphic to the qubit Pauli-operator algebra via the Jordan-Wigner transformation [2], codes based on the two algebras have different notions of locality and thus qualitatively different physical interpretations.
  • Bosonic code — Bosonic (fermionic) codes are associated with bosonic (fermionic) degrees of freedom.
