Spin cat code[1]

Description

An analogue of the two-component cat code designed for atomic ensembles. Was designed by using the Holstein-Primakoff mapping [2] (see also [3]) to pull back the phase-space structure of a bosonic system to the compact phase space of a quantum spin.

Parent

  • Single-spin code

Cousin

  • Two-component cat code — The Spin-cat code construction utilizes the Holstein-Primakoff mapping [2] (see also [3]) to convert various expressions for GKP states into codes for spin systems.
