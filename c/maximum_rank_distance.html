Maximum-rank vector code

Name

Maximum-rank vector code[1][2]

Description

Linear rank-metric code with maximum rank \([n,k,n-k+1]_q\) over rank metric. For any vector \(g=(g_1, \ldots, g_n)\) with linearly independent elements \(g_i\) over \(GF(q)\), the generator matrix \begin{align} G = \begin{pmatrix} g_1^{[0]} & g_2^{[0]} & \ldots & g_n^{[0]} \\ g_1^{[1]} & g_2^{[1]} & \ldots & g_n^{[1]} \\ g_1^{[k-1]} & g_2^{[k-1]} & \ldots & g_n^{[k-1]} \end{pmatrix}~, \end{align} where \([i]:=q^i\) and \(k=n-d+1\) defines a maximum rank distance code.

Protection

\(d\)-distance codes protect agains all errors of rank not greater \(\lfloor (d-1)/2\rfloor\)

Parent

  • Vector code — Vector rank-metric codes with maximum rank.

Cousin

  • Reed-Solomon code — The construction of the generators matrix of both codes uses a Vandermonde matrix.
