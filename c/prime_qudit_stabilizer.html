Prime-qudit stabilizer code

Name
Prime-qudit stabilizer code[1]
Description
An \(((n,p^k,d))_p\) (with \(p\) prime) stabilizer code is denoted as \([[n,k]]_p\) or \([[n,k,d]]_p\), where \(d\) is the code's distance. Logical subspace is the joint \(+1\) eigenspace of a set of \(p^{n-k}\) commuting qudit Pauli operators forming the code's stabilizer group, which cannot contain \(-I\). The distance is the minimum weight of an error that implements a nontrivial logical operation on the code.
Protection
Detects errors on up to \(d-1\) qudits, and corrects erasure errors on up to \(d-1\) qudits. More generally, define the normalizer \(\mathsf{N(S)}\) of \(\mathsf{S}\) to be the set of all operators that commute with all \(S\in\mathsf{S}\). A stabilizer code can correct a generalized Pauli error set \({\mathcal{E}}\) if and only if \(E^\dagger F \notin \mathsf{\hat{N}(S)}\setminus \mathsf{S}\) for all \(E,F \in {\mathcal{E}}\).
Parent(s)

  • Galois-qudit stabilizer code (A prime-qudit stabilizer code is a Galois-qudit stabilizer code with \(m=1\).)
Cousin(s)
  • Qudit surface code (Translation-invariant prime-qudit stabilizer groups with local generators and with code distance being the linear system size are equivalent to a finite number of copies of the qudit surface code stabilizer group via a local constant-depth Clifford circuit [2].)
