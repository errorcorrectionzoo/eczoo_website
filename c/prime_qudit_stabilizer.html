\(\mathbb{Z}_p\)-qudit stabilizer code

Name
\(\mathbb{Z}_p\)-qudit stabilizer code[1] 
Description
An \(((n,p^k,d))_p\) (with \(p\) prime) stabilizer code is denoted as \([[n,k]]_p\) or \([[n,k,d]]_p\), where \(d\) is the code's distance. Logical subspace is the joint \(+1\) eigenspace of a set of \(p^{n-k}\) commuting qudit Pauli operators forming the code's stabilizer group, which cannot contain \(-I\). The distance is the minimum weight of an error that implements a nontrivial logical operation on the code.
Protection
Detects errors on up to \(d-1\) qudits, and corrects erasure errors on up to \(d-1\) qudits. The normalizer \(N(S)\) of the stabilizer \(S\) is the set that contains all operators that commutes with \(S\). The stabilizer code can correct errors \({\mathcal{E}}\) if and only if \(\hat{E}^\dagger \hat{F} \notin \hat{N}(S)\setminus \hat{S}\) for all \(E,F \in {\mathcal{E}}\).
