Five-qudit code[1][2]

Description

Modular-qudit \([[5,1,3]]_{q}\) stabilizer code that generalizes the five-qubit perfect code using properties of the multiplicative group \(\mathbb{Z}_q\) [1]; see also [2; Thm. 13]. It has four stabilizer generators consisting of \(X Z Z^\dagger X^\dagger I\) and its cyclic permutations.

Protection

Protects against a single error on any one qudit. Detects two-qudit errors.

Cousins

  • Five-qubit perfect code — The five-qudit code reduces to the five-qubit perfect code for \(q=2\).
  • Braunstein five-mode code — The Braunstein five-mode code is a bosonic analogue of the five-qudit code.
  • Five-rotor code — The five-rotor code is a bosonic analogue of the five-qudit code.
