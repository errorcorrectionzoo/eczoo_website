\(E_8\) Gosset lattice code[1]

Description

BW lattice in dimension \(8\), which is the lattice corresponding to the \([8,4,4]\) Hamming code via the mod-two lattice construction.

Protection

The \(E_8\) lattice has a nominal coding gain of \(2\). It exhibits the densest lattice packing [2] and packing [3] in eight dimensions.'

Parents

Cousins

  • \([7,4,3]\) Hamming code — The \([8,4,4]\) extended Hamming code yields the \(E_8\) lattice code via the mod-two lattice construction [4; Ex. 10.5.2].
  • Octacode — The octacode yields the \(E_8\) lattice code via Construction \(A_4\) [5][6].
  • \(q\)-ary Hamming code — The \([4,2,3]_3\) ternary Hamming code can be used to obtain the \(E_8\) root lattice code [4; Ex. 10.5.5].
  • \(E_8\) Gosset lattice-shell code
