Generalized color code[1]

Description

Member of a family of non-Abelian topological codes, defined by a finite group \( G \), that serves as a generalization of the color code (for which \(G=\mathbb{Z}_2\)).

Parents

  • Group-based quantum code
  • Topological code — Quantum-double code Hamiltonians admit topological phases associated with finite groups \(G\).

Cousins

  • Quantum-double code — Generalized color code for group \(G\) on the 4.8.8 lattice is equivalent to a \(G\) quantum double model and another \(G/[G,G]\) quantum double model defined using the abelianization of \(G\).
  • Color code — The generalized color code for \(G=\mathbb{Z}_2\) reduces to the color code.

References

[1]
C. G. Brell, “Generalized color codes supporting non-Abelian anyons”, Physical Review A 91, (2015). DOI; 1408.6238
