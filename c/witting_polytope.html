Witting polytope code

Description

Spherical \((8,240,1)\) code whose codewords are the vertices of the Witting complex polytope and the \(4_{21}\) real polytope. This code is optimal and unique up to equivalence [1][2] and forms a tight spherical 7-design [2; Ch. 14]. Codewords form the minimal lattice-shell code of the \(E_8\) lattice.

A representation of the codewords consists of all 112 permutations of the four vectors \((0,0,0,0,0,0,\pm 2,\pm 2)\) along with the 128 vectors \((\pm 1)^{\times 8}\). See [3; pg. 132] for a complex representation.

Protection

Code yields an optimal solution to the kissing problem in 4D [4][5].
