Algebraic-geometry (AG) code[1][2][3]

Description

Binary or \(q\)-ary code constructed from an algebraic curve of some genus over a finite field via the evaluation construction, the residue construction, or more general constructions that yield nonlinear codes. Linear AG codes from the first two constructions are also called geometric Goppa codes.

In alternative conventions (not used here), AG codes are often restricted to be linear and/or include evaluation codes defined using algebraic varieties.

Rate

A class of linear AG codes with non-constant genera exceed the asymptotic Gilbert-Varshamov (GV) bound [4] (see also Ref. [5]). Roughly speaking, this breakthrough result implies that AG codes can outperform random codes. Such families of codes are optimal. Generally, the rate of AG codes satisfies \begin{align} \frac{k}{n} \geq 1 - \frac{d}{n} - \frac{1}{\sqrt{q}-1}~, \end{align} which comes from the Drinfeld-Vladut bound [6]. There exist several sequences of algebraic curves that attain this bound and yield asymptotically good codes [7][8] (see Ref. [9] for details). Nonlinear AG codes can beat even this bound.

Notes

See book by Goppa [10].
