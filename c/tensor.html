Tensor code

Description

Martix-based code constructed out of two linear binary or \(q\)-ary codes \(C_A,C_B\) in an outer-product construction denoted as \(C_A \otimes C_B\). The codewords are those matrices whose column vectors are in \(C_A\) and whose row vectors are in \(C_B\).

Notes

Tensor codes formed by two random linear codes are robustly testable [1][2].
