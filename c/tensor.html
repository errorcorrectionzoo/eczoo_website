Tensor-product code[1][2][3][4]

Description

Also called a tensor code or a product code. A matrix-based code constructed out of two linear binary or \(q\)-ary codes \(C_A,C_B\) in an outer-product construction denoted as \(C_A \otimes C_B\). The codewords are those matrices whose column vectors are in \(C_A\) and whose row vectors are in \(C_B\). For linear codes \(C_A=[n_A,k_A,d_A]\) and \(C_B=[n_B,k_B,d_B]\), the resulting tensor code is \(C_A \otimes C_B=[n_A n_B,k_A k_B,d_A d_B]\).

Protection

Tensor codes can be useful for protecting against burst errors [5][6]. Tensor codes formed by two random linear codes are robustly testable [7][8].

Notes

See book [9], Ch. 18 for an exposition.
