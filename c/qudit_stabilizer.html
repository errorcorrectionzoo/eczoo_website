Modular-qudit stabilizer code

Name
Modular-qudit stabilizer code[1]
Description

An \(((n,K,d))_q\) modular-qudit code whose logical subspace is the joint \(+1\) eigenspace of a set of \(|{\mathsf{S}}|\) commuting qudit Pauli operators forming the code's stabilizer group \({\mathsf{S}}\), which cannot contain \(e^{i \phi} I\) for any \(\phi \neq 0\). The distance \(d\) is the minimum weight of a qudit Pauli string that implements a nontrivial logical operation on the code.

A modular-qudit stabilizer code encoding an integer number of qudits (\(K=q^k\)) is denoted as \([[n,k]]_{q}\) or \([[n,k,d]]_{q}\). For composite \(q\), such codes need not encode an integer number of qudits, with \(K=q^n/|\mathsf{S}|\) [2]. This is because \(|{\mathsf{S}}|\) need not be a power of \(q\), as group generators may have different orders. Prime-qudit stabilizer codes, where \(q=p\) for some prime \(p\), do not suffer from this issue and encode \(n-k\) logical qudits, with \(K=p^{n-k}\).

Each code can be represented by a stabilizer generator matrix \(H=(H_X|H_Z)\), where each row \((h_X|h_Z)\) is the \(q\)-ary symplectic representation of a stabilizer generator. (Standard form goes here.)

Protection
Detects errors on up to \(d-1\) qudits, and corrects erasure errors on up to \(d-1\) qudits. More generally, define the normalizer \(\mathsf{N(S)}\) of \(\mathsf{S}\) to be the set of all operators that commute with all \(S\in\mathsf{S}\). A stabilizer code can correct a Pauli error set \({\mathcal{E}}\) if and only if \(E^\dagger F \notin \mathsf{N(S)}\setminus \mathsf{S}\) for all \(E,F \in {\mathcal{E}}\).
Decoder
Trellis decoder for prime-dimensional qudits [3].
Notes
A standardized definition of the qudit stabilizer group is developed in [2].

  • Double-semion code (Double-semion code can be realized as a modular-qudit stabilizer code with \(q=4\) [4].)

Cousin(s)
  • Stabilizer code (Modular-qudit stabilizer codes for prime-dimensional qudits \(q=p\) inherit most of the features of qubit stabilizer codes, including encoding an integer number of qudits and a Pauli group with a unique number of generators. Conversely, qubit stabilizer codes can be extended to qudits by decorating the appropriate Pauli strings with powers, and the local-translational-invariant (LDI) form [5] of the generator matrix yields systematic extensions to prime qudits.)
  • Modular-qudit surface code (Translation-invariant 2D prime-qudit topological stabilizer codes are equivalent to a finite number of copies of the qudit surface code stabilizer group via a local constant-depth Clifford circuit [6][7].)
  • Galois-qudit stabilizer code (Recalling that \(q=p^m\), Galois-qudit stabilizer codes can also be treated as prime-qudit stabilizer codes on \(mn\) qudits, giving \(k=nm-r\) [8]. The case \(m=1\) reduces to conventional prime-qudit stabilizer codes on \(n\) qudits.)
  • Abelian topological code (All abelian topological orders can be realized as modular-qudit stabilizer codes [4].)
