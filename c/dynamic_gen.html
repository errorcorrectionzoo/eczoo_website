Dynamically-generated quantum error-correcting code

Name

Dynamically-generated quantum error-correcting code[1]

Description

Code whose natural definition is in terms of a many-body scaling limit of a local dynamical process. Such processes update the code structure and can include random unitary evolution or non-commuting projective measurements.

Parent

  • Subsystem quantum error-correcting code

Children

  • Floquet code
  • Random-circuit code

Cousins

  • Clifford-deformed surface code (CDSC) — To create CDSCs, a dynamical process is applied on top of the surface code.
  • Quantum low-density parity-check (QLDPC) code — QLDPC codes can arise form a dynamical process [2].
