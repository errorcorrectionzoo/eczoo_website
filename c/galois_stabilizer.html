Galois-qudit stabilizer code

Name
Galois-qudit stabilizer code[1]
Description
An \(((n,p^k,d))_{GF(q)}\) stabilizer code is denoted as \([[n,k]]_{GF(q)}\) or \([[n,k,d]]_{GF(q)}\), where \(d\) is the code's distance. Stub.
