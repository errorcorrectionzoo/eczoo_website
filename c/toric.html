Kitaev's toric code

Name
Kitaev's toric code [1]  
Description
A family of stabilizer codes whose generators are few-body \(X\)-type and \(Z\)-type Pauli strings associated to the stars and plaquettes, respectively, of a tessellation of a two-dimensional surface (with a qubit located at each edge of the tesselation). Toric often refers to the specific code construction on the two-dimensional torus, the construction on surfaces with boundaries is often called the planar code [2], and surface code is often an alternative name for the general construction.
Protection
When defined on an \(L\times L\) square tiling of the torus, protects against \(L\) errors. More generally, the code distance is related to the homology of the tessellation [3].
Decoder
Minimum weight perfect-matching [3][4], union-find [5], renormalization group [6][7], tensor network [8], Markov chain Monte Carlo [9], cellular automaton [10], machine learning [11][12][13].
fault_tolerance
Fault-tolerant threshold of \(0.5-1.1\%\) depending on the error model [14].
gates
Clifford gates can be implemented via lattice surgery [15][16] or braiding defects [17][18][19][20]. Non-Clifford gates require magic state distillation [21] or just-in-time decoding [22].
threshold
Depolarizing noise: \(18.9\%\) [23], independent \(X/Z\) errors: \(10.9\%\) [24], loss errors: \(50\%\) [25] (all for square tiling).
encoder
Unitary [26][27][28], dissipative [29], stabilizer measurement [30].
Notes
When treated as ground states of the code Hamiltonian, the code states realize \(\mathbb{Z}_2\) topological order, a topological phase of matter that also exists in \(\mathbb{Z}_2\) lattice gauge theory [31].
Realizations
Distance-two toric codes have been implemented by Andersen et al. [32], Erhard et al. [33] and Google Quantum AI [34].
Parent(s)
  • Calderbank-Shor-Steane (CSS) code (Plaquette and star operators are stabilizer generators.)
  • Homological code
  • Quantum low-density parity-check (LDPC) code (Stabilizer generator weights and qubit degrees are given by the properties of the tessellation.)
