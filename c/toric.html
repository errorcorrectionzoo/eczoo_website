Kitaev's toric code

Name
Kitaev's toric code [1]    
Description
A family of stabilizer codes whose generators are few-body \(X\)-type Pauli strings (stars) and few-body \(Z\)-type Pauli strings on any surface plaquette operators on an arbitrary lattice on an arbitrary orientable 2D surface [1].
Decoder
MWPM, union find
Features
(fault_tolerance)
Fault-tolerant threshold around \(\sim 10\%\).
