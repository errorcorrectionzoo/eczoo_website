XY surface code[1]

Description

Non-CSS derivative of the surface code whose generators are \(XXXX\) and \(YYYY\), obtained by mapping \(Z \to Y\) in the surface code.

Protection

As a stabilizer code, \([[n=O(d^2), k=O(1), d]]\).

Code Capacity Threshold

\(50\%\) at infinite \(Z\) bias with maximum-likelihood decoder [2].\(18.7\%\) for standard depolarising noise with maximum-likelihood decoder [2].

Threshold

\(6.32(3)\%\) for infinite \(Z\) bias, and thresholds of \(\sim 5\%\) for \(Z\) bias around \(\eta = 100\) using a variant of the minimum-weight perfect matching decoder CITE arXiv:1907.02554.

Parent

  • Clifford-deformed surface code (CDSC) — XY code is obtained from the surface code by applying \(H\sqrt{Z}H\) to all qubits, thereby exchaning \(Z\leftrightarrow Y\).
