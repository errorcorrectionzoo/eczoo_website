Griesmer code[1][2][3]

Description

A \([n,k,d]\) binary or \(q\)-ary linear code is a Griesmer code if parameters \(n\), \(k\), \(d\), and \(q\) are such that the Griesmer bound \begin{align} n\geq\sum_{j=0}^{k-1}\left\lceil \frac{d}{q^{j}}\right\rceil ~, \end{align} where \(\left\lceil x\right\rceil \) is the ceiling function, becomes an equality.

An \([n,2,d]_q\) code exists if and only if it is not excluded by the Griesmer bound. Every Griesmer code is generated by its minimum-weight codewords [4].

Parent

  • Maximum distance separable (MDS) code — Singleton bound implies the Griesmer bound.

Children

  • Denniston code
  • Simplex code — Simplex codes saturate the Griesmer bound ([5], Exer. 5.1.11).

Cousins

  • Divisible code — Griesmer codes over prime fields are divisible [6].
  • Anticode — Several anticode (e.g., [7][8]) and related [9] constructions saturate the Griesmer bound; see Refs. [5][10][11] for more details.
  • Binary BCH code — The \([15,5,7]\) BCH code extended with a parity check saturates the Griesmer bound ([5], pg. 157).
  • Grassmannian code — The binary Grassmannian \([35,6,16]\) code, whose points lie on the Grassmannian \({\mathbb{G}(2,4)}\), attains the Griesmer bound [12].
  • Projective RM (PRM) code — PRM codes for \(r=1\) attain the Griesmer bound for all \(m\) [12].
  • Projective geometry code — Acrs corresponding to Griesmer codes are called Griesmer arcs ([13], pg. 248). There is a one-to-one correspondence between Griesmer codes and minihypers [14][15]; see Ref. [16], Sec. 14.2.4 and Ref. [17] for more details.
