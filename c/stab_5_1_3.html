\([[5,1,3]]\) code

Name
\([[5,1,3]]\) code[1]
Description
Five-qubit stabilizer code with generators that are symmetric under cyclic permutation of qubits, \begin{align} \begin{split} S_1 &= IXZZX \\ S_2 &= XZZXI \\ S_3 &= ZZXIX \\ S_4 &= ZXIXZ. \end{split}\end{align}
Protection
Smallest stabilizer code that protects against a single error on any one qubit. Detects two-qubit errors.
Transversal Gates
Pauli gates are transversal.
Realizations
First realized in NMR [2].Demonstration with superconducting qubits [3].
Parent(s)
  • Stabilizer code (\([[5,1,3]]\) code is the smallest stabilizer code to correct a single qubit error.)
Cousin(s)
  • Perfect quantum code (The smallest perfect code.)
  • Quantum cyclic code (\([[5,1,3]]\) code is the smallest known example of quantum cyclic code .)
  • Hamiltonian-based code (\([[5,1,3]]\) code Hamiltonian is local when expressed in terms of Majorana operators [4].)
  • Majorana stabilizer code (\([[5,1,3]]\) code Hamiltonian is local when expressed in terms of Majorana operators [4].)
  • Pastawski-Yoshida-Harlow-Preskill (HaPPY) code (The \([[5,1,3]]\) encoding isometry tiles various holographic codes because its corresponding tensor is perfect [5].)
  • \([[4,2,2]]\) CSS code (\([[4,2,2]]\) can be derived from \([[5,1,3]]\) code using a protocal that converts \([[n,k,d]]\) code into an \([[n-1, k+1, d-1]]\) code; see Sec. 3.5 in Gottesman [6].)
