Self-dual linear code

Name
Self-dual linear code 
Description
A binary linear code \(C\) whose dual code \(C^\perp = \{ y\in\mathbb{Z}_2^{\times n} ~|~ x\cdot y=0 \forall x\in C\}\) is equal to \(C\).
Protection
The generator matrix of \(C^\perp\) is the parity check matrix of \(C\), and visa versa.
