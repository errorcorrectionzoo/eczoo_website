Anticode[1][2]

Description

Code for which the distance between any two codewords is less than or equal to some value \(\delta\) called the maximum distance. Anticodes can be used to construct codes that saturate the Griesmer bound; see Refs. [3][4][5] for more details.

Parent

  • Binary code

Cousins

  • Griesmer code — Several anticode (e.g., [6][7]) and related [8] constructions saturate the Griesmer bound; see Refs. [3][4][5] for more details.
  • Projective geometry code — There is a relation between anticodes and minihypers ([3], pg. 295).
