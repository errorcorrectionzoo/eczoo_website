Quantum Tanner code[1]

Description

Stub.

Rate

Asymptotically good QLDPC codes.

Decoding

Linear-time decoder similar to the small-set-flip decoder for quantum expander codes [2].Linear-time decoder [3].

Realizations

Used to obtain explicit lower bounds in the sum-of-squares game [4].States that, on average, achieve small violations of check operators for quantum Tanner codes require a circuit of non-constant depth to make. They are used in the proof of the No low-energy trivial states (NLTS) conjecture [5].

Notes

For details, see talk by A. Leverrier.

Parents

Child

  • Rotated surface code — Specializing the quantum Tanner construction to the surface code yields the rotated surface code [6].

Cousins

  • Tanner code — Tanner codes are used in constructing quantum Tanner codes.
  • Tensor code — Tensor codes are used in constructing quantum Tanner codes.
  • Expander lifted-product code — Quantum Tanner codes are an attempt to construct asymptotically good QLDPC codes that are similar to but simpler than expander lifted-product codes; see Ref. [3] for connection between the codes.
