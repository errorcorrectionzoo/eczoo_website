Qudit code

Name
Qudit code 
Description
Encodes \(K\)-dimensional Hilbert space into a \(q^n\)-dimensional (\(n\)-qudit) Hilbert space. Usually denoted as \(((n,K))_q\) or \(((n,K,d))_q\), whenever the code's distance \(d\) is defined.
Parent(s)
  • Quantum error-correcting code (QECC) in finite dimensions
Parent of

Cousin of
