120-cell code

Description

Spherical \((4,600,1/(\phi^2\sqrt{2})\) code (with \(\phi\) the golden ratio) whose codewords are the vertices of the 120-cell. The code forms a spherical 11-design. See [1; Table 1][2] for a realization of the 600 codewords.

Realizations

Improved proofs of the Bell-Kochen-Specker (BKS) theorem [1].

Parent

  • Polytope code

Child

  • 600-cell code — Vertices of a 120-cell can be split up into vertices of five 600-cells [2][1].

References

M. Waegell and P. K. Aravind, “Parity Proofs of the Kochen–Specker Theorem Based on the 120-Cell”, Foundations of Physics 44, 1085 (2014). DOI; 1309.7530

H. S. M. Coxeter. Regular polytopes. Courier Corporation, 1973.
