Reed-Solomon (RS) code[1][2]

Description

An \([n,k,n-k+1]_q\) linear code based on polynomials over \(GF(q)\). Let \(\{\alpha_1,\cdots,\alpha_n\}\) be \(n\) distinct nonzero elements of \(GF(q)\) with \(q>n\). Encodes \(\mu=\{\mu_0,\cdots,\mu_{k-1}\}\) into \(\{f_\mu(\alpha_1),\cdots,f_\mu(\alpha_n)\}\), with polynomial \begin{align} f_\mu(x)=\mu_0+\mu_1 x + \cdots + \mu_{k-1}x^{k-1}. \end{align}

Protection

Corrects erasures on \(n-k\) coordinates.

Decoding

Berlekamp-Welch decoder [3], assuming that \(t \geq (n+k)/2\).

Realizations

Numerous applications: computer hard-disks, DVDs, digital audio tapes, 3G telephone networks (IS-2000, Release D), NASA deep-space exploration, etc.Correcting pooled testing results for SARS-CoV-2 [4].

Notes

Decoding Reed-Solomon codes is equivalent to the problem of Polynomial Reconstruction and has applications to cryptography [5].

Parents

Cousins

  • Hamming code
  • Cyclic code — Reed-Solomon codes are cyclic whenever the length divides \(q-1\).
  • Approximate secret-sharing code — The classical information in this code is encoded using a Reed-Solomon code.
  • Bose–Chaudhuri–Hocquenghem (BCH) code
  • Convolutional code — Convolutional codes are often used in concatenation with Reed-Solomon codes for communication [7].
  • Galois-qudit polynomial code (QPyC) — Polynomial codes are CSS codes constructed from Reed-Solomon codes.
  • Maximum-rank vector code — The construction of the generators matrix of both codes uses a Vandermonde matrix.
  • Prime-qudit polynomial code (QPyC) — Polynomial codes are CSS codes constructed from Reed-Solomon codes.
  • Rank-modulation code — RS codes can be used to design rank modulation codes [8].
