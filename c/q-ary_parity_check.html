\(q\)-ary parity-check code

Description

Also known as a sum-zero code. An \([n,n-1,2]_q\) linear \(q\)-ary code whose codewords consist of the message string appended with a parity-check digit such that the sum over all coordinates of each codeword is zero.

Parents

  • Reed-Solomon (RS) code — RS codes for \(k=n-1\) are parity-check codes [1].
  • Cyclic linear \(q\)-ary code — Since permutations preserve coordinate sums, the cyclic permutation of a parity-check codeword is another codeword.
  • Maximum distance separable (MDS) code

Cousin

  • Parity-check code
