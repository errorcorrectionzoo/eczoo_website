Cyclic code

Name
Cyclic code 
Description
A code \(C\) of length \(n\) over an alphabet is cyclic if, for each string \(c_1 c_2 \cdots c_n\in C\), the cyclically shifted string \(c_n c_1 \cdots c_{n-1}\in C\).
