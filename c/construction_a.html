Mod-2 lattice code[1]

Description

Lattice constructed from a binary linear \([n,k,d]\) code using Construction \(A_2\) [2]. Each binary codeword \(c\) of the code is mapped to an infinite set of points \(x\) such that \(x = c\) modulo two. Linearity of the code ensures that the resulting set of points forms a lattice.
