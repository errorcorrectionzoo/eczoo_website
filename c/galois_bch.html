Galois-qudit BCH code[1][2][3][4][5][6]

Description

True Galois-qudit stabilizer code constructed from BCH codes via either the Hermitian construction or the Galois-qudit CSS construction.

Parent

  • True Galois-qudit stabilizer code — Galois-qudit BCH codes constructed via the CSS construction are Galois-qudit CSS codes, and the rest are true stabilizer codes.

Cousins

  • Qubit BCH code
  • Bose–Chaudhuri–Hocquenghem (BCH) code
  • Galois-qudit CSS code — Some Galois-qudit BCH codes are CSS.
