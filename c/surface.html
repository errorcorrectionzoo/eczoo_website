Kitaev's surface code

Name
Kitaev's surface code[1][2]
Description
A family of CSS stabilizer codes whose generators are few-body \(X\)-type and \(Z\)-type Pauli strings associated to the stars and plaquettes, respectively, of a tessellation of a two-dimensional surface (with a qubit located at each edge of the tesselation). Toric code often either refers to the construction on the two-dimensional torus or is an alternative name for the general construction. The construction on surfaces with boundaries is often called the planar code [3].
Protection
When defined on an \(L\times L\) square tiling of the torus, protects against \(L\) errors. More generally, the code distance is related to the homology of the tessellation [4].
Encoder
Unitary [5][6][7].Dissipative [8].Stabilizer measurement-based [9].
TransversalÂ Gates
Pauli gates on torus geometry.
Gates
Clifford gates can be implemented via lattice surgery [10][11] or braiding defects [12][13][14][15].Non-Clifford gates require magic state distillation [16] or just-in-time decoding [17].
Decoder
Minimum weight perfect-matching [4][18], union-find [19], renormalization group [20][21], tensor network [22], Markov chain Monte Carlo [23], cellular automaton [24], machine learning [25][26][27].
Threshold
\(10.9\%\) with independent \(X/Z\) errors for square tiling [28].\(18.9\%\) with depolarizing noise for square tiling [29].\(50\%\) with loss errors for square tiling [30].\(3.3\%\) with phenomenological noise for square tiling [31].\(0.5-1.1\%\) for various error models [32].
Realizations
Distance-two surface codes have been implemented by Andersen et al. [33], Erhard et al. [34], and Google Quantum AI [35].Signatures of corresponding topological phase of matter detected in superconducting circuits [36] and two-dimensional arrays of Rydberg atoms [37].Distance-three surface code implemented at ETH Zurich [38].
Notes
2D and 3D surface code visualization tool.Toric code introductions from error-correction perspective by J. Haah and condensed-matter perspective by M. Levin and C. Nayak.
Parent(s)
  • Calderbank-Shor-Steane (CSS) stabilizer code (Plaquette and star operators are stabilizer generators.)
  • Topological code (When treated as ground states of the code Hamiltonian, the code states realize \(\mathbb{Z}_2\) topological order, a topological phase of matter that also exists in \(\mathbb{Z}_2\) lattice gauge theory [39].)
Cousin(s)
  • Quantum low-density parity-check (QLDPC) code (Stabilizer generator weights and qubit degrees are given by the properties of the tessellation, and most tesselations yield QLDPC codes.)
  • Hypergraph product code (Planar (toric) code obtained from hypergraph product of two repetition (cyclic) codes.)
  • Shor's \([[9,1,3]]\) code (Shor's code is the smallest surface code defined on the projective plane [40].)
  • \([[4,2,2]]\) CSS code (\([[4,2,2]]\) code is the smallest toric code.)
