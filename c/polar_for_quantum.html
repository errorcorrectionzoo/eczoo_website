Quantum-classical polar code[1][2]

Description

Polar code used to transmit classical information over quantum channels.

Rate

Codes achieve the symmetric Holevo information for sending classical information over any quantum channel.

Decoding

Quantum-limited successive-cancellation (SC) joint-detection receiver [1].

Parent

  • Linear binary code

Child

  • Polar code

Cousin

  • Binary phase-shift keyring (BPSK) — BPSK concatenated with quantum-classical polar codes achieves the Holevo capacity for the pure-loss channel [3].
