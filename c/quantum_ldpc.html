Quantum low-density parity-check (QLDPC) code

Name
Quantum low-density parity-check (QLDPC) code
Description
Family of \([[n,k,d]]\) stabilizer codes for which the number of qubits participating in each stabilizer generator and the number of stabilizer generators that each qubit participates in are both bounded by a constant as \(n\to\infty\).
Protection
Detects errors on \(d-1\) qubits, corrects errors on \(\left\lfloor (d-1)/2 \right\rfloor\) qubits. Asymptotic scaling of \(k\) and \(d\) with \(n\) is often of interest, and this depends heavily on the code construction. A good quantum LDPC code family satisfies \(k \in \theta(n) \) and \(d \in \theta(n)\) [1], and the first examples of good codes are a family of lifted-product codes. A geometrically local LDPC code is an LDPC code where the qubits involved in any syndrome bit are contained in a fixed volume that does not scale with \(n\); such codes are limited to having \(d=O(n^{1-1/D})\) for \(D\)-dimensional lattice geometries [2].
Threshold
Quantum LDPC codes with a constant encoding rate can reduce the overhead of fault-tolerant quantum computation to be constant [3].

Parent of
  • Ramanujan tensor-product code (\(Z\)- and \(X\)- row and column weights are bounded from above by the weights of the two codes that are used in the construction of this code [4].)
  • Quantum expander code
  • Homological product code (Homological product construction is a primary tool for generating QLDPC codes with favorable parameters.)
  • XYZ product code (Using the assumption that the input classical codes are LDPC codes, XYZ-product codes will be QLDPC codes.)
  • Distance-balanced code (Distance-balancing procedure can yield QLDPC codes; see Thm. 1 in Ref. [5].)
Cousin(s)
  • Low-density parity-check (LDPC) code
  • Hamiltonian-based code (Codespace is the ground-state space of a local code Hamiltonian, consisting of an equal linear combination of stabilizer generators.)
  • Dynamically-generated quantum error-correcting code (QLDPC codes can arise form a dynamical process [6].)
  • Color code (Stabilizer generator weights and qubit degrees are given by the properties of the tessellation, and most tesselations yield QLDPC codes.)
  • Honeycomb code (The Floquet check operators are weight-two, and each qubit participates in one check each round.)
  • Kitaev surface code (Stabilizer generator weights and qubit degrees are given by the properties of the cellulation, and most cellulations yield QLDPC codes.)
  • Distance-balanced code (Distance-balancing procedure can yield QLDPC codes; see Thm. 1 in Ref. [5].)
  • Topological code (Stabilizer-based topological codes (e.g., the surface code) on appropriate tesselations are QLDPC.)
