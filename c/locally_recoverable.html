Locally recoverable code (LRC)

Description

A code \(C\) over \(\mathbb{F}_q^{n}\) with length \(n\) is \(r\)-locally recoverable, or have locality \(r\), if \(\forall i \in [n]\), \(\exists I_i \subset [n]\setminus i\)  s.t. \(|I_i|\leq r\) and the projection of the set \(\mathcal{C}(i,a)=\{x\in C : x_i=a\}\) on to the coordinates in \(I_i\), i.e., \(\mathcal{C}_{I_i}(i,a)\) is disjoint from \(\mathcal{C}_{I_i}(i,a^\prime)\) where \(a\neq a^\prime\). In other words, given a codeword \(x\) and any coordinate \(i\), \(x_i\) can be recovered from at most \(r\) other coordinates of \(x\). This definition can be generalized to \(t\)-LRC code, if every coordinate is recoverable from \(t\) disjoint subsets of coordinates. An \(r\)-locally recoverable code with length \(n\) and dimension \(k\) is denoted as (n,k,r) LRC code.

Protection

The distance, \(d\), of (n,k,r) LRC codes satisfies \begin{align} d\leq n-k-\left \lceil\frac{k}{r}\right \rceil+2 \end{align} It should be obvious from definition that \(r\leq k\). Moreover for \(k=r\), the bound on distance gives Singleton bound. A study on parameters of LRC(t-LRC) codes together with known bounds can be found in [1].

Rate

The rate of (n,k,r) LRC codes satisfies \begin{align} \frac{k}{n}\leq\frac{r}{r+1} \end{align}

Notes

Generalized Singleton bound does not account for \(q\)-ary alphabet size. A more generalized bound(including non-linear case) is given in [2]
