Elliptic code

Description

Evaluation AG code on a set of points lying on an elliptic curve, i.e., a curve of genus one.

Parent

  • Evaluation AG code — Elliptic codes are evaluation AG codes with \(\cal X\) being an elliptic curve, i.e., curve of genus one ([1], Ch. 3.2; [2]).

Cousin

  • Maximum distance separable (MDS) code — Elliptic codes can be MDS ([3], Ex. 15.5.3; [1], pg. 310; [4], Sec. 4.4.2).
