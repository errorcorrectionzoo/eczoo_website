Two-component cat code[1]

Description

Code whose codespace is spanned by two coherent states \(\left|\pm\alpha\right\rangle\) for nonzero complex \(\alpha\). An orthonormal basis for the codespace consists of the bosonic cat states \begin{align} |\overline{\pm}\rangle=\frac{\left|\alpha\right\rangle \pm\left|-\alpha\right\rangle }{\sqrt{2\left(1\pm e^{-2|\alpha|^{2}}\right)}} \end{align} for any complex \(\alpha\).

A closely related approximate cat code is called T4C code [2].

Protection

Two-legged cat codes for large \(\alpha\) provide protection against modal dephasing, i.e., diffusion of the angular degree of freedom of the mode. They do not protect against photon loss events, but there exist modifications based on sign alternation [3], squeezing [4][5][5], and detuning [6] that can add such protection.

Encoding

Lindbladian-based dissipative encoding [7] utilizing two-photon absorption [8][9][10][11]. Encoding passively protects against cavity dephasing, suppressing dephasing noise exponentially with \(|\alpha|^2\) [7].Hamiltonian-based 'Kerr-cat' encoding utilizing the Kerr-effect Hamiltonian [12] (see also Ref. [13]).Colored dissipation [14].Combined dissipative and Hamiltonian-based encoding utilizing two-photon exchange with an ancillary qubit [15].Critical encoding at non-zero detuning [16].

Gates

Universal gates in the quantum optical setting can be performed using teleportation, Bell measurements, displacements, and rotations [17]. An earlier protocol requires a non-linear interaction and uses state teleportation [18].Universal gates in the microwave setting can be performed using displacement operators and a rotation based on the Kerr nonlinearity [7]. Kerr nonlinearity converts coherent states into Yurke-Stoler states [19].Bias-preserving Hamiltonian-based CNOT gate utilizing an \(X\) gate with a topological Berry phase [20][21]. Bias-preserving SWAP gate [22].

Fault Tolerance

Fault-tolerant error-correction procedure using small amplitude coherent states [23].Bias-preserving Hamiltonian-based CNOT gate [21] is part of a universal bias-preserving gate set that can be made fault tolerant using concatenation [20][21].Ancilla qubits encoded in two-component cat codes yield fault-tolerant syndrome extraction circuits [24].

Realizations

Lindbladian-based [25][26] and Hamiltonian-based 'Kerr-cat' [27] encodings have been achieved in superconducting circuit devices by the Devoret group; Ref. [26] also demonstrated a displacement-based gate. The Lindbladian-based scheme has further achieved a suppression of bit-flip errors that is exponential in the average photon number [28] up to a bit-flip time of 1ms. A bit-flip time of 1s has been achieved in a similar system in the classical bit regime [29].T4C code realized in a superconducting circuit device by the Wang group [2].

Notes

Pedagogical introduction to cat codes in the context of microwave cavities can be found in Refs. [30][31], and in the context of optical systems in books [32][33][34].
