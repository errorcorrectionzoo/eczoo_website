Quantum BCH code[1][2][3]

Description

Stabilizer code constructed from a self-orthogonal binary BCH code via the CSS construction, from a Hermitian self-orthogonal quaternary BCH code via the stabilizer-over-\(GF(4)\) construction, or by taking a Euclidean self-orthogonal BCH code over \(GF(2^m)\), converting it to a binary code, and applying the CSS construction. Other work uses Hermitian [4] or Euclidean [5] (Galois-qudit) self-orthogonal BCH codes.

Parent

  • Qubit stabilizer code — QBCH codes constructed via the CSS construction are CSS codes, and the rest are stabilizer codes over \(GF(4)\).

Cousins

  • Binary BCH code — Binary BCH codes are used to construct a subset of QBCH codes via the CSS construction.
  • Bose–Chaudhuri–Hocquenghem (BCH) code — BCH codes are used to construct QBCH codes via the CSS and stabilizer-over-\(GF(4)\) constructions.
  • Calderbank-Shor-Steane (CSS) stabilizer code — Binary BCH codes are used to construct a subset of QBCH codes via the CSS construction.
  • Stabilizer code over \(GF(4)\) — Hermitian self-orthogonal linear quaternary BCH codes are used to construct a subset of QBCH codes via the stabilizer-over-\(GF(4)\) construction.
