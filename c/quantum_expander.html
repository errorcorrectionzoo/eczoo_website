Quantum expander code

Name
Quantum expander code[1]
Description
CSS codes constructed from bipartite expander graphs with bounded left and right vertex degrees. For every bipartite graph there is an associated matrix (the parity check matrix) with columns indexed by the left vertices, rows indexed by the right vertices, and 1 entries whenever a left and right vertex are connected. This matrix can serve as the parity check matrix of a classical code. Two bipartite expander graphs can be used to construct a quantum CSS code (the quantum expander code) by using the parity check matrix of one as \(X\) checks, and the parity check matrix of the other as \(Z\) checks.
Protection
Pauli errors of weight \(\leq t\), distance scales as \(\Omega(n^{1/2})\).
Rate
\([[n,k=\Theta(n),d=O(\sqrt{n})]]\) code with asymptotically constant rate.
Decoder
Ref. [1] details a linear time decoder, which corrects \(\Omega(n^{1/2})\) adversarial errors.
Threshold
Current estimate of \(2.7 \cdot 10^{-16}\) in locally stochastic noise model [2].
Parent(s)
  • Quantum low-density parity-check (QLDPC) code
Cousin(s)
  • Expander code (Quantum expander codes are constructed from classical expander codes.)
