Higher-dimensional surface code[1]

Description

A family of Kitaev surface codes on planar or toric surfaces of dimension greater than two. Stub.

Decoding

Improved BP-OSD decoder [2].

Threshold

Phenomenological noise model for the 4D toric code: \(4.3\%\) under improved BP-OSD decoder [2].

Cousins

  • Fractal surface code — Fractal surface codes are obtained by removing qubits from the surface code on a cubic lattice.
  • Self-correcting quantum code — The 4D toric code is a self-correcting quantum memory [1][3].
