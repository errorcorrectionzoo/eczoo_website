Matrix-product code[1]

Description

Code constructed using a concatenation procedure that yields a code consisting of all products of codewords in \(M\) length-\(n\) \(q\)-ary codes and an \(M\times N\) \(q\)-ary matrix with \(N\geq M\). A prominent subclass is the case with \(A\) is non-singular by columns (NSC).

Decoding

Decoder up to half of the minimum distance for NSC codes [2].

Parent

  • Galois-field \(q\)-ary code

Child

  • Generalized Reed-Muller (RM) code — Applying a special case of the matrix-product procedure yields GRM codes [1].

Cousin

  • True Galois-qudit stabilizer code — Hermitian self-orthogonal matrix-product codes can be used to construct stabilizer codes over \(GF(q^2)\) [3][4].
