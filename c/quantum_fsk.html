Coherent FSK (CFSK) c-q code[1][2]

Description

Bosonic c-q code encoding into coherent states that are frequency-shifted with certain initial relative phase. Codewords are coherent states \(|\alpha_m\rangle\), where \begin{align} \alpha_m = \alpha e^{i(\omega_0+[m-1]\Delta\omega)t+i(m-1)\Delta\theta} \end{align} for common frequency \(\omega_0\), frequency shift \(\Delta\omega < 2\pi/T\), total time \(T\), and phase shift \(\Delta\theta\).

Decoding

Bondurant receiver [3].Cyclic receiver [4].Time-resolving receiver [1][2][5].Bayesian inference [1].

Realizations

Time-resolving quantum receiver [5].Bondurant receiver [4].Bayesian inference [2].
