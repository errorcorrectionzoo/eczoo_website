Low-density generator-matrix (LDGM) code

Description

Binary or \(q\)-ary linear code with a sparse generator matrix. More precisely, a member of an infinite family of \([n,k,d]\) codes for which the number of nonzero entries in each row and column of the generator matrix are both bounded by a constant as \(n\to\infty\). The dual of an LDGM code has a sparse parity-check matrix and is called an LDPC code.

Rate

Certain LDGM codes come close to achieving Shannon capacity [1].
