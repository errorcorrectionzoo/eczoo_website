Bose–Chaudhuri–Hocquenghem (BCH) code[1]

Description

Stub.

Notes

See database [2] for explicit codes.See corresponding MinT database entry [3].

Parents

  • Generalized Reed-Solomon (GRS) code — BCH codes are subfield subcodes of GRS codes.
  • Cyclic binary code
  • Perfect code — Double error-correcting BCH codes are quasi-perfect [4] (see also [5], Ch. 9).

Children

  • Hamming code — Binary Hamming codes are primite binary BCH codes [3]. All binary Hamming codes are cyclic; see Ref. [6], Thm. 12.22.
  • Reed-Muller (RM) code — RM\((r,m)\) codes are subcodes of BCH codes of designed distance \(2^{m-r}-1\) ([5], Ch. 13).

Cousins

  • Reed-Solomon (RS) code — Cyclic RS codes of length \(q-1\) are a particular case of BCH codes; see Remark 15.3.21 of Ref. [7].
  • Skew-cyclic code
