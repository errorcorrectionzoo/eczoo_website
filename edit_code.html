Error correction zoo — edit code

Thanks for contributing! On this page you can edit the information relating to a code in the error correcting zoo database.

Please note that changes are not saved as you type; it is recommended that you prepare your text content separately, in your favorite text editor, and then paste your text in the fields below. Once you're ready, you can follow the instructions at the bottom of the page to submit your contribution. Please allow for some time for us to process your submission.

You can use some selected LaTeX commands in text fields, as explained here.
